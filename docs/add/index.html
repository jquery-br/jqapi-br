.add()

Categories: Traversing > Miscellaneous Traversing

.add( seletor ) Retorna: jQuery

Descrição: Adiciona elementos no conjunto de elementos encontrados.

  • adicionado na versão: 1.0.add( seletor )

    seletorUma string representando um seletor para encontrar elementos adicionais para adicionar ao conjunto de elementos encontrados.

  • adicionado na versão: 1.0.add( elementos )

    elementosUm ou mais elementos para adicionar ao conjunto de elementos encontrados.

  • adicionado na versão: 1.0.add( html )

    htmlUm fragmento HTML para adicionar ao conjunto de elementos encontrados.

  • adicionado na versão: 1.4.add( seletor, contexto )

    seletorUma string representando um seletor para encontrar elementos adicionais para adicionar ao conjunto de elementos encontrados.

    contextoO ponto no documento no qual o seletor deve começar a procurar; similar ao argumento contexto do método $(seletor, contexto).

Dado um objeto jQuery que representa um conjunto de elementos DOM, o método .add() constrói um novo objeto jQuery a partir da união destes elementos com os elemtnos passados como argumento do método. O argumento para o método .add() pode praticamente qualquer coisa que o $() aceita, incluindo um seletor jQuery, referência para elementos DOM, ou um fragmento HTML.

O conjunto atualizado de elementos pode ser usado num método seguinte(em cadeia), ou atribuído a uma variável para ser usado depois. Por exemplo:

$("p").add("div").addClass("widget");
var pdiv = $("p").add("div");

O seguinte não vai salvar os elementos adicionados, porque o método .add() cria um novo conjunto e deixa o conjunto original em pdiv sem modificações:

var pdiv = $("p");
pdiv.add("div");  // ERRADO, pdiv não vai mudar

Considere uma página com uma lista simples e um parágrafo após ela:

<ul>
  <li>list item 1</li>
  <li>list item 2</li>
  <li>list item 3</li>
</ul>
<p>a paragraph</p>

Nós podemos selecionar os itens da lista e então selecionar o parágrafo usando um seletor ou uma referência para o elemento DOM como argumento para o método .add():

$('li').add('p').css('background-color', 'red');

Or:

$('li').add(document.getElementsByTagName('p')[0])
  .css('background-color', 'red');

O resultado desta chamada vai ser um fundo vermelho atrás de todos os quatro elementos. Usando um fragmento HTML como argumento do método .add()(assim como na terceira versão), nós podemos criar elementos adicionais em tempo de execução e adicionar estes elementos ao conjunto encontrado de elementos. Digamos, por exemplo, que queremos alterar o fundo do itens da lista junto ao de um parágrafo recém criado:

$('li').add('<p id="new">new paragraph</p>')
  .css('background-color', 'red');

Apesar do novo parágrafo ter sido criado e sua cor de fundo ter mudado, ele ainda não aparece na página. Para colocá-lo na página, nós podemos adicionar um dos métodos de inserção na cadeia de métodos.

A partir do jQuery 1.4 os resultados do método .add() vão sempre ser retornados na ordem do documento(ao invés de uma simples concatenação).

Exemplos:

Exemplo: Acha todas as divs e faz uma borda. Então adiciona todos os parágrafos ao objeto jQuery para deixar seu fundo amarelo. 

<!DOCTYPE html>
<html>
<head>
  <style>
 div { width:60px; height:60px; margin:10px; float:left; }
 p { clear:left; font-weight:bold; font-size:16px; 
     color:blue; margin:0 10px; padding:2px; }
 </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div></div>

  <div></div>
  <div></div>
  <div></div>
  <div></div>
  <div></div>

  <p>Added this... (notice no border)</p>
<script>

$("div").css("border", "2px solid red")
        .add("p")
        .css("background", "yellow");
</script>

</body>
</html>

Exemplo: Adiciona mais elementos, criados na execução, ao conjunto de elementos encontrados. 

<!DOCTYPE html>
<html>

Exemplo: Adiciona um ou mais Elementos ao conjunto de elementos encontrados. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p><span id="a">Hello Again</span>
<script>$("p").add(document.getElementById("a")).css("background", "yellow");</script>

</body>
</html>

Exemplo: Demonstra como adicionar elementos ao uma coleção existente. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p><span id="a">Hello Again</span>
<script>var collection = $("p");
// capture the new collection
collection = collection.add(document.getElementById("a"));
collection.css("background", "yellow");</script>

</body>
</html>
