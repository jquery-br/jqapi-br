.ajaxComplete()

Categoria: Ajax > Ajax Event Handlers Global

.ajaxComplete( handler(event, XMLHttpRequest, ajaxOptions) ) Retorno: jQuery

Descrição: Registra um handler para ser chamado quando a requisição Ajax completar. Isso é um evento Ajax.

  • adicionado na versão: 1.0.ajaxComplete( handler(event, XMLHttpRequest, ajaxOptions) )

    handler(event, XMLHttpRequest, ajaxOptions) A função a ser invocada.

Sempre que uma requisição Ajax completar, o jQuery vai disparar o evento ajaxComplete. Todo e qualquer handler que foi registrado com o método .ajaxComplete() será executado.

Para ver esse método em ação, podemos configurar uma requisição Ajax básica:

<div class="trigger">Trigger</div>
<div class="result"></div>
<div class="log"></div>

Podemos anexar nosso evento handler a qualquer elemento:

$('.log').ajaxComplete(function() {
  $(this).text('Triggered ajaxComplete handler.');
});

Agora podemos fazer uma requisição Ajax usando qualquer método do jQuery:

$('.trigger').click(function() {
  $('.result').load('ajax/test.html');
});

Quando o usuário clica no botão e a requisição Ajax completa, a mensagem de log é mostrada.

Nota: Devido ao .ajaxComplete() ser implementado como função da instância de um objeto jQuery, podemos usar a palavra privada this como fazemos aqui, para referenciar o elemento selecionado dentro da função de callback.

Todos os handlers ajaxComplete são invocados, independentemente de qual requisição Ajax que foi completada. Se precisamos diferenciar entre as requisições, podemos usar parametros passados ao handler. Para cada hora que um handler ajaxComplete é executado, é passado o evento do objeto, o objeto XMLHttpRequest , e as definições do objeto que foi usado na criação da requisição. Por exemplo, podemos restringir nossa callback para a apenas dar handler se o objeto tiver uma URL em particular:

Nota: Você pode pegar o conteúdo do retorno do Ajax através do xhr.responseXML ou xhr.responseHTML para xml e html respectivamente.

$('.log').ajaxComplete(function(e, xhr, settings) {
  if (settings.url == 'ajax/test.html') {
    $(this).text('Triggered ajaxComplete handler. The result is ' + xhr.responseHTML);
  }
});

Mostra uma mensagem quando a requisição Ajax completa.

   $(this).append("<li>Request Complete.</li>");
