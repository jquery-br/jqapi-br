.ajaxError()

Categorias: Ajax > Global Ajax Event Handlers

.ajaxError( handler(event, jqXHR, ajaxSettings, thrownError) ) Retorna: jQuery

Descrição: Registra um handler a ser chamado quando requisições Ajax é concluída com erro. É um Evento Ajax .

  • adicionado na versão: 1.0 .ajaxError( handler(event, jqXHR, ajaxSettings, thrownError) )

    handler(event, jqXHR, ajaxSettings, thrownError) A função a ser invocada.

Sempre que uma requisição Ajax finaliza com um erro, jQuery dispara o evento ajaxError. Qualquer e todos handlers que tenham sido registrados com o método .ajaxError() são executados nesse momento.

Para observar esse método em ação, criamos uma requisição básica de carga Ajax.

<button class="trigger">Trigger</button>
<div class="result"></div>
<div class="log"></div>

Inclua o event handler a qualquer elemento:

$("div.log").ajaxError(function() {
  $(this).text( "Triggered ajaxError handler." );
});

Agora, faça uma requisição Ajax usando qualquer método JQuery:

$("button.trigger").click(function() {
  $("div.result").load( "ajax/missing.html" );
});

Quando o usuário clica no botão e a requisição Ajax falha, porque o arquivo requisitado não está presente, a mensagem de log é exibida.

Nota: Porque .ajaxError() é implementado como um método de instâncias do Jquery, você pode usar a palavra reservada this dentro de uma função callback para referenciar os elementos selecionados.

Todos handlers ajaxError são invocados, independente de qual requisição Ajax tenha sido concluída. Para diferenciar entre as requisições, você pode usar os parâmetros passados ao handler. Cada vez que um handler ajaxError é executados, lhe é passado o objeto event, o objeto jqXHR (versões anteriores ao jQuery 1.5, o objeto XHR ), e o objeto settings que foi usado na criação da requisição. Se a requisição falhou porque o JavaScript lançou uma exceção, o objeto exception é passado ao handler como o quarto parâmetro. Por exemplo, para restringir o retorno de erro de manipulação de eventos para lidar com uma URL em particular:

$( "div.log" ).ajaxError(function(e, jqxhr, settings, exception) {
  if ( settings.url == "ajax/missing.html" ) {
    $(this).text( "Triggered ajaxError handler." );
  }
});

Exibe uma mensagem quando a requisição Ajax falha.

$("#msg").ajaxError(function(event, request, settings){
  $(this).append("<li>Error requesting page " + settings.url + "</li>");
});
