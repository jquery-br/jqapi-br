.ajaxStart()

Categories: Ajax > Global Ajax Event Handlers

.ajaxStart( handler() ) Returns: jQuery

Descrição: Registra um handler que será chamado quando a primeira requisição Ajax se iniciar. Isso é um evento do Ajax.

  • Desde a versão: 1.0.ajaxStart( handler() )

    handler()A função que será invocada.

Sempre que uma requisição ajax está para ser enviada, o jQuery checa se existem outras requisições Ajax pendentes. Se não há nenhuma em progresso, o jQuery dispara o evento ajaxStart. Todos e quaisquer handlers que tenham sido registrados com o método .ajaxStart() são executados neste momento.

Para observar este método em ação, podemos configurar uma requisição Ajax básica:

<div class="trigger">Trigger</div>
<div class="result"></div>
<div class="log"></div>

Nós podemos vincular nosso handler do evento a qualquer elemento:

$('.log').ajaxStart(function() {
  $(this).text('Triggered ajaxStart handler.');
});

Agora podemos fazer uma requisição Ajax usando qualquer método jQuery:

$('.trigger').click(function() {
  $('.result').load('ajax/test.html');
});

Quando o usuário clicar no botão e a requisição Ajax for enviada, a mensagem de log será mostrada.

Nota: Como .ajaxStart() é implementado como um método de instância jQuery, nós podemos usar a palavra-chave this para referenciar aos elementos selecionados a partir da função de callback.

Exemplo:

Mostrar uma mensagem de carregamento sempre que uma requisição Ajax começar (e não houver outra já em curso).

$("#loading").ajaxStart(function(){
   $(this).show();
 });
