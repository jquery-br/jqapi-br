.ajaxSuccess()

Categories: Ajax > Global Ajax Event Handlers

.ajaxSuccess( handler(event, XMLHttpRequest, ajaxOptions) ) Retorna: jQuery

Descrição: Associa uma função para ser executada sempre que uma requisição Ajax se completar com sucesso. Isso é um evento do Ajax.

  • adicionado na versão: 1.0.ajaxSuccess( handler(event, XMLHttpRequest, ajaxOptions) )

    handler(event, XMLHttpRequest, ajaxOptions)A função que será invocada.

Sempre que uma requisição Ajax se completa com sucesso, o jQuery dispara o evento ajaxSuccess. Todos e quaisquer handlers que foram registrados pelo método .ajaxSuccess() são executados nesse momento.

Para observar este método em ação, podemos configurar uma requisição Ajax básica:

<div class="trigger">Trigger</div>
<div class="result"></div>
<div class="log"></div>

Nós podemos vincular nosso handler do evento a qualquer elemento:

$('.log').ajaxSuccess(function() {
  $(this).text('Triggered ajaxSuccess handler.');
});

Agora podemos fazer uma requisição Ajax usando qualquer método jQuery:

$('.trigger').click(function() {
  $('.result').load('ajax/test.html');
});

Quando o usuário clicar no botão e a requisição Ajax se completar com sucesso, a mensagem de log será mostrada.

Nota: Como .ajaxStop() é implementado como um método de instância jQuery, nós podemos usar a palavra-chave this para referenciar aos elementos selecionados a partir da função de callback.

Todos os handlers ajaxSuccess são invocados, independentemente de qual requisição Ajax terminou. Se precisamos diferenciar entre as requisições, podemos usar os parametros passados ao handler. Cada vez que um handler ajaxSuccess é executado, lhe é passado um event object, o objeto XMLHttpRequest, e o objeto de configuração que foi usado pra criar a requisição. For exemplo, podemos restringir nosso callback a só manipular requisições feitas a uma determinada URL:

Nota: Você pode obter o conteúdo da resposta Ajax na propriedades xhr.responseXML ou xhr.responseHTML para XML e HTML, respectivamente.

$('.log').ajaxSuccess(function(e, xhr, settings) {
  if (settings.url == 'ajax/test.html') {
    $(this).text('Triggered ajaxSuccess handler. The ajax response was:' 
                     + xhr.responseHTML );
  }
});
