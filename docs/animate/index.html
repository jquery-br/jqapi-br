.animate()

Categorias: Efeitos > Personalizado

.animate( propriedades, [ duração ], [ efeito_transição ], [ função_finalizada ] ) Retornar para: jQuery

Descrição: Realiza um animação personalizada de um conjunto de propriedades e elementos CSS.

  • adicionado na versão: 1.0 .animate( propriedades, [ duração ], [ efeito_transição ], [ função_finalizada ] )

    propriedades: Um conjunto de propriedades CSS que será utilizado no decorrer da animação.

    duração: Uma string ou um número determinando a duração da animação.

    efeito_transição: Uma string indicando qual função/efeito de transição que será utilizado.

    função_finalizada: Uma função a qual será chamada quando a animação estiver completa.

  • adicionado na versão: 1.0 .animate( propriedades, opções )

    propriedades: Um conjunto de propriedades CSS que será utilizado no decorrer da animação.

    opções: Um conjunto de opções adicionais que será passado para o método de transição. Chaves suportadas:

    • duração: Uma string ou um número determinando a duração da animação.
    • efeito_transição: Uma string indicando qual função/efeito de transição será utilizado.
    • função_finalizada: Uma função a qual será chamada quando a animação estiver completa.
    • passo-a-passo: Uma função que será chamada a cada passo da animação.
    • fila: Um valor Boolean indicando quando colocar a animação na fila. Se false, a animação irá começar imediatamente.
    • efeito_transição_especial: Um conjunto de uma ou mais propriedades CSS definidos pelo parâmetro propriedades e suas correspondentes funções de transição (adicionado 1.4).

A função .animate() nos permite criar efeitos de animação em qualquer elemento CSS. O único parâmetro obrigatório é um conjunto de propriedades CSS. Este conjunto é similar ao que pode ser enviado na função .css() , exceto que o número de propriedades é mais restrito.

Valores e Propriedades de Animação

Todas as propriedades animadas devem ser animadas para uma váriavel numérica, exceto a regra a seguir: a maioria da propriedades que não são númericas não podem ser animadas utilizando a funcionalidade básica do jQuery. (Por exemplo, width, height, ou left podem ser animadas, mas background-color não pode.) Os valores de propriedades são tratadas como um número de pixels a não ser quando especificado. As unidades em e % podem ser especificadas quando aplicavéis.

Além das propriedades do tipo style, algumas propriedades personalizadas como scrollTop e scrollLeft podem ser animadas.

Propriedades CSS de Taquigrafia (ex: margin, background, border) não são suportadas. Por exemplo, se você deseja obter a propriedade margin renderizada, utilize: $(element).css('marginTop') e $(element).css('marginRight'), e assim por adiante.

Além das propriedades númericas, cada propriedade pode utilizar as strings 'show', 'hide', e 'toggle'. Estes atalhos permitem realizar funções personalizadas de esconder e mostrar que levem em conta o tipo de exibição do elemento.

Propriedades animadas podem ser relativas. Se um valor é fornecido utilizando um conjunto de caracteres += ou -= , então o valor alvo é computado adicionando ou subtraindo o número fornecido do número atual da propriedade.

Ao contrário das animações como .slideDown() e .fadeIn(), a função .animate() não faz os elementos ficarem visivéis como parte do efeito. Por exemplo, utilizando a função $('someElement').hide().animate({height:'20px'}, 500}), a animação será realizada, mas o elemento continuará escondido.

Duração

A duração é realizada utilizando milisegundos; valores maiores significam animações mais demoradas, não mais rápidas. As strings 'fast' e 'slow' podem ser fornecidas para indicar uma duração de 200 e 600 milisegundos respectivamente.

Animação Completa (função_finalizada)

Se fornecida, a função de resposta função_finalizada é realizada quando o processo de animação está completo. Isto pode ser muito útil para juntar várias animações em sequência. A função de retorno função_finalizada não envia nenhum argumento, porém referência this pode ser utilizado para obter o elemento DOM que está sendo animado. Se múltiplos elementos são animados, a função de retorno é executada uma vez para cada elemento, e não uma vez para a animação toda.

Uso Básico

Para animar qualquer elemento como uma simples imagem:

<div id="clickme">
  Click here
</div>
<img id="book" src="book.png" alt="" width="100" height="123"
  style="position: relative; left: 10px;" />

Para animar as propriedades da imagem opacity, a propriedade left offset, e a propriedade height simultâneamente:

$('#clickme').click(function() {
  $('#book').animate({
    opacity: 0.25,
    left: '+=50',
    height: 'toggle'
  }, 5000, function() {
    // Animation complete.
  });
});

Note que o valor da propriedade height é 'toggle'. Como a imagem era visível anteriormente, a animação encolhe a altura para 0 escondendo a imagem. Um segundo clique realizaria a ação inversa:

A propriedade opacity da imagem já está no seu valor final, então esta propriedade não é animada no segundo clique. Assim como o valor final da propriedade left é um valor relativo, a imagem move-se mais para a direita durante a animação do segundo clique.

Propriedades de efeitos de direção (top, right, bottom, left) não possuem um efeito aplicável nos elementos se sua propriedade position é static, o qual é por padrão.

O projeto jQuery UI extende a função .animate() permitindo alguns estilos não númericos como cores serem animados. O projeto também inclui mecanismos para especificar animações através de classes CSS ao invés de propriedades individuais.

Função Passo-a-Passo

A segunda versão da função .animate() permite a utilização de uma função passo-a-passo — é um função chamada a cada passo da animação. Esta função é muito útil para habilitar alguns efeitos de animação personalizado ou alterar a animação a medida que está sendo realizada. A função aceita dois argumentos (now e fx), e a referência this é pode se utilizado para obter o elemento DOM que esta sendo animado.

  • now: o valor númerico da propriedade que esta sendo animada a cada passo
  • fx: uma referência para o objeto padrão jQuery.fx, o qual contém um número de propriedades como elem para o elemento animado, start e end para o primeiro e para o último valor da propriedade animada respectivamente, e prop para a propriedade que está sendo animada.

Note que a função passo-a-passo é chamada para cada propriedade animada em cada elemento DOM animado. Por exemplo, em dois items de uma lista(li), a função passo-a-passo é ativada 4 vezes a cada passo da animação: 

$('li').animate({
  opacity: .5,
  height: '50%'
},
{
  step: function(now, fx) {
    var data = fx.elem.id + ' ' + fx.prop + ': ' + now;
    $('body').append('<div>' + data + '</div>');
  }
});

Efeito de transição

O último parâmetro da função .animate() é uma string chamada efeito_transição. Uma função de efeito de transição específica a velocidade da animação em diferentes pontos da mesma animação. As únicas implementações na biblioteca do jQuery são, uma chamada swing, e uma que progride em ritmo constante chamada linear. Mais funções de efeitos de transição podem ser utilizadas utilizando plug-ins, o mais notável é o jQuery UI suite.

Efeito de Transição-por-Propriedade

A partir da versão 1.4 do jQuery, podemos setar efeitos de transição-por-propriedade dentro de uma única chamada da função .animate(). Na primeira versão da função .animate(), cada propriedade pode pegar um array como seu valor: O primeiro membro do array é a propriedade CSS e o segundo membro é uma função de transição. Se uma função de transição-por-propriedade não é definida para uma propriedade particular, ele utiliza o valor opcional de transição da função .animate(). Se o efeito de transição não está definido, a função padrão swing é utilizada.

Por exemplo, para animar simultâneamente as propriedades width e height com a função de transição swing e a propriedade opacity com a função de transição linear:

$('#clickme').click(function() {
  $('#book').animate({
    width: ['toggle', 'swing'],
    height: ['toggle', 'swing'],
    opacity: 'toggle'
  }, 5000, 'linear', function() {
      $(this).after('<div>Animation complete.</div>');
  });
});

Na segunda versão da função .animate(), o conjunto de opções incluem a propriedade specialEasing, o qual é um conjunto de propriedades CSS e suas funções de transição correspondentes. Por exemplo, para simultâneamente animar a propriedade width utilizando o efeito de transição linear e a propriedade height utilizando a função de transição easeOutBounce:

$('#clickme').click(function() {
  $('#book').animate({
    width: 'toggle',
    height: 'toggle'
  }, {
    duration: 5000,
    specialEasing: {
      width: 'linear',
      height: 'easeOutBounce'
    },
    complete: function() {
      $(this).after('<div>Animation complete.</div>');
    }
  });
});

Como mencionado anteriormente, um plugin é necessário para a animação easeOutBounce funcionar.

Notas Adicionais:

  • Todos os efeitos do jQuery, incluindo a função .animate(), podem ser desativados globalmente configurando a variável jQuery.fx.off = true, o que efetivamente troca o valor de duração para 0. Para mais informações, veja jQuery.fx.off.
