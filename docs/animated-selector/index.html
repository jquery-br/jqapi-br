:animated Selector

Categorias: Selectors > Basic Filter

animated selector

adicionado na versão: 1.2jQuery(':animated')

Descrição: Seleciona todos os elementos que estão em progresso de animação no momento em que o seletor é executado.

Exemplo:

Troca a cor de qualquer div que está sendo animada.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background:yellow; border:1px solid #AAA; width:80px; height:80px; margin:0 5px; float:left; }
  div.colored { background:green; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="run">Run</button>

  <div></div>
  <div id="mover"></div>
  <div></div>
<script>

    $("#run").click(function(){
      $("div:animated").toggleClass("colored");
    });
    function animateIt() {
      $("#mover").slideToggle("slow", animateIt);
    }
    animateIt();
</script>

</body>
</html>
