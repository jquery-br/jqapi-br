.appendTo()

Categorias: Manipulação > DOM Insertion, Inside

.appendTo( target ) Retorna: jQuery

Descrição: Insere cada elemento da coleção encontrada ao final do elemento alvo ( target ).

  • adicionado na versão: 1.0.appendTo( target )

    targetUm seletor, elemento, string HTML ou objeto jQuery; os elementos encontrados serão inseridos ao final do(s) elemento(s) especificados por este parâmetro.

O métodos.append() e .appendTo() executam a mesma tarefa. A unica diferença é a sintaxe, onde se coloca o conteúdo e o elemento. Com o .append(), o seletor que precede o método será o container onde o conteúdo será inserido. Com o .appendTo(), ao contrário, o conteúdo precede o método, sendo uma expressão seletora ou uma marcação criada dinamicamente, sendo assim inserida no container alvo.

Considere o seguinte HTML:

<h2>Greetings</h2>
							<div class="container">
							<div class="inner">Hello</div>
							<div class="inner">Goodbye</div>
							</div>

Nós podemos criar o conteúdo e inserí-lo em vários elementos, ao mesmo tempo:

$('<p>Test</p>').appendTo('.inner');

Each inner <div> element gets this new content:

<h2>Greetings</h2>
							<div class="container">
							<div class="inner">
							Hello
							<p>Test</p>
							</div>
							<div class="inner">
							Goodbye
							<p>Test</p>
							</div>
							</div>

Nós podemos também selecionar um elemento na página e inserí-lo em outro:

$('h2').appendTo($('.container'));

Se um elemento selecionado dessa forma for inserido em outro lugar, ele será movido para o destino (não copiado):

<div class="container">
							<div class="inner">Hello</div>
							<div class="inner">Goodbye</div>
							<h2>Greetings</h2>
							</div>

Entretanto, se existir mais de um elemento de destino, cópias do elemento inserido serão criadas para cada elemento de destino após o primeiro.

Exemplo:

Insira todos os spans no elemento com o ID "foo"

<!DOCTYPE html>
							<html>
							<head>
							<style>#foo { background:yellow; }</style>
							<script src="http://code.jquery.com/jquery-git.js"></script>
							</head>
							<body>
							<span>I have nothing more to say... </span>

							<div id="foo">FOO! </div>
							<script>$("span").appendTo("#foo"); // check append() examples</script>

							</body>
							</html>
