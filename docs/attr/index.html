.detach()

Categorias: Manipulação > Remoção DOM

.detach( [ seletor ] ) Retorna: jQuery

Descriçaõ: Remove todos os elementos selecionados da DOM.

  • Versão Adicionada: 1.4.detach( [ seletor ] )

    seletorum selector e uma expressão utilizando para filtrar os conjuntos de elementos para a remoção.

O método .detach() é semelhante ao .remove(), exceto que o .detach mantem todo documento JQuery associado ao elemento. Esse método é util quando queremos remover elementos que podem ser reinsersidos no DOM mais tarde.

Example:

Detach todos os parágrafos do DOM.

<!DOCTYPE html>
<html>
<head>
  <style>p { background:yellow; margin:6px 0; } p.off { background: black; }</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p> 
  how are 
  <p>you?</p>
  <button>Attach/detach paragraphs</button>
<script>
    $("p").click(function(){
      $(this).toggleClass("off");
    });
    var p;
    $("button").click(function(){
      if ( p ) {
        p.appendTo("body");
        p = null;
      } else {
        p = $("p").detach();
      }
    });</script>

</body>
</html>
