Attribute Contains Prefix Selector [name|="value"]

Categorias: Selectors > Attribute

attributeContainsPrefix selector

adicionado na versão: 1.0jQuery('[attribute|="value"]')

  • attribute
    Um nome de atributo.
    value
    Um valor para o atributo. Aspas são obrigatórias.

Descrição: Seleciona elementos que têm o atributo especificado com um valor quer seja passado por uma string ou começando com essa string seguida de um hífen (-).

Este seletor foi adicionado à especificação CSS para manipular atributos de idioma.

Exemplo:

Encontra todos links com um atributo hreflang que seja inglês.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <a href="example.html" hreflang="en">Some text</a> 

  <a href="example.html" hreflang="en-UK">Some other text</a>
  
<script>$('a[hreflang|="en"]').css('border','3px dotted green');</script>

</body>
</html>
