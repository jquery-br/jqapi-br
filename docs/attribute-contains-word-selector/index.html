Attribute Contains Word Selector [name~="value"]

Categorias: Selectors > Attribute

attributeContainsWord selector

adicionado na versão: 1.0jQuery('[attribute~="value"]')

  • attribute
    Nome do atributo.
    value
    Um valor para o atributo. Aspas são obrigatórias.

Descrição: Seleciona elementos que tem o atributo especificado com um valor contendo uma dada palavra, delimitado por espaços.

Este seletor corresponde a string de teste encontrando cada palavra no atributo valor, onde uma "palavra"é definida como uma string delimitada por espaço em branco. O seletor é encontrado se a string de teste é exatamente igual a qualquer uma das palavras.

Exemplo:

Encontra todos inputs com um atributo name que contenha a palavra 'man' e atribui o valor com algum texto.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <input name="man-news" />

  <input name="milk man" />
  <input name="letterman2" />
  <input name="newmilk" />
<script>$('input[name~="man"]').val('mr. man is in it!');</script>

</body>
</html>
