Attribute Ends With Selector [name$="value"]

Categorias: Selectors > Attribute

attributeEndsWith selector

adicionado na versão: 1.0jQuery('[attribute$="value"]')

  • attribute
    Nome do atributo.
    value
    Um valor para o atributo. Aspas são obrigatórias.

Descrição: Seleciona elementos que tem o atributo especificado com um valor contendo uma dada palavra. A comparação é case sensitive.

Exemplo:

Encontra todos inputs com um atributo name que termina com 'letter' e insere texto neles.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <input name="newsletter" />

  <input name="milkman" />
  <input name="jobletter" />
<script>$('input[name$="letter"]').val('a letter');</script>

</body>
</html>
