Attribute Starts With Selector [name^="value"]

Categorias: Selectors > Attribute

attributeStartsWith selector

adicionado na versão: 1.0jQuery('[attribute^="value"]')

  • attribute
    Nome do atributo.
    value
    Um valor para o atributo. Aspas são obrigatórias.

Descrição: Seleciona elementos que possuem o atributo especificado com um valor começando exatamente com uma dada string.

Este seletor pode ser útil para a identificação de elementos em páginas produzidas por frameworks server-side que produzem HTML com IDs de elementos sistemáticos. No entanto, será mais lento do que usando seletor de classes influenciando assim classes, se você puder, agrupe como elementos.

Exemplo:

Encontra todos inputs com um antributo name que começa com 'news'e insere texto neles.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <input name="newsletter" />

  <input name="milkman" />
  <input name="newsboy" />
<script>$('input[name^="news"]').val('news here!');</script>

</body>
</html>
