.before()

Categorias: Manipulation > DOM Insertion, Outside

.before( content, [ content ] ) Returns: jQuery

Descrição: Insere o conteúdo, especificado pelo parâmetro, antes de cada elemento da coleção resultante.

  • version added: 1.0.before( content, [ content ] )

    contentHTML string, elemento DOM ou objeto jQuery à inserir antes de cada elemento encontrado.

    contentUm ou mais elementos DOM adicionais, arrays de elementos, HTML strings ou objetos jQuery A inserir antes de cada elemento encontrado.

  • version added: 1.4.before( function )

    functionUma função que retornará um HTML string que será inserido antes de cada elemento encontrado.

Os métodos .before() e .insertBefore() executam a mesma tarefa. A grande diferença está na sintaxe, na localização do conteúdo e do alvo. No .before(), a expressão seletora que precede o método é o container onde o conteúdo será inserido anteriormente. Já no .insertBefore(), por outro lado, o conteúdo precede o método, ambos em forma de seletor ou expressão ou como marcação criada dinamicamente, sendo inserido imediatamente antes ao elemento alvo.

Considere o HTML à seguir:

<div class="container">
                <h2>Greetings</h2>
                <div class="inner">Hello</div>
                <div class="inner">Goodbye</div>
              </div>

Você pode criar o conteúdo e inserí-lo antes de cada elemento de uma única vez:

$('.inner').before('<p>Test</p>');

Cada elemento inner <div> terá esse novo conteúdo:

<div class="container">
                <h2>Greetings</h2>
                <p>Test</p>
                <div class="inner">Hello</div>
                <p>Test</p>
                <div class="inner">Goodbye</div>
              </div>

Você pode, também, selecionar um elemento na página e inserí-lo antes de outro:

$('.container').before($('h2'));

Se o elemento selecionado dessa forma for inserido noutro lugar, ele será movido antes ao alvo (e não clonado):

<h2>Greetings</h2>
              <div class="container">
                <div class="inner">Hello</div>
                <div class="inner">Goodbye</div>
              </div>

Se houver mais de um elemento alvo selecionado, no entanto, cópias do elemento inserido serão criadas para cada alvo após o primeiro.

A partir do jQuery 1.4, .before() e .after() também funcionarão em 'disconnected DOM nodes':

$("<div/>").before("<p></p>");

O resultado será um jQuery set que contém um parágrafo e uma div (nesta ordem).

Argumentos adicionais

Similar aos outros métodos de inserção de conteúdo como .prepend() e .after(), .before() também suporta a passagem de múltipos parâmetroscomo entrada. As entradas suportadas incluem DOM elements, objetos jQuery, HTML strings, e arrays de de elementos DOM.

Por exemplo, O código a seguir irá inserir duas novas <div>s e uma existente <div> ante de cada parágrafo:

var $newdiv1 = $('<div id="object1"/>'),
              newdiv2 = document.createElement('div'),
              existingdiv1 = document.getElementById('foo');

              $('p').first().before($newdiv1, [newdiv2, existingdiv1]);

Já que o .before() aceita vários parâmetros adicionais, o mesmo resultado pode ser alcançado passando as três <div>s como três argumentos separadamente, como a seguir: $('p').first().before($newdiv1, newdiv2, existingdiv1). O tipo de número de argumentos dependerá da forma que você coletar os elementos em seu código.
