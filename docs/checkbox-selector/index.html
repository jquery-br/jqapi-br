:checkbox Selector

Categorias: Selectors > Form

checkbox selector

Adicionado na versão: 1.0 jQuery(':checkbox')

Descrição: Seleciona todos os elementos do tipo checkbox.

$(':checkbox') é equivalente à $('[type=checkbox]'). Como os outros pseudo-class selectors (aqueles que começam com ":") é recomendavel utilizar-lo com alguma tag ou outro selector; caso o contrário, o universal selector ("*") é aplicado. Em outras palavras, $(':checkbox') é equivalente à $('*:checkbox'), então $('input:checkbox') seria uma melhor alternativa.

Exemplo:

Procurar todos checkbox inputs.

<!DOCTYPE html>
<html>
<head>
  <style>
  textarea { height:25px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>
    <input type="button" value="Input Button"/>
    <input type="checkbox" />

    <input type="checkbox" />
    <input type="file" />
    <input type="hidden" />

    <input type="image" />
    <input type="password" />
    <input type="radio" />

    <input type="reset" />
    <input type="submit" />
    <input type="text" />

    <select><option>Option<option/></select>
    <textarea></textarea>
    <button>Button</button>
  </form>

  <div>
  </div>
<script>

    var input = $("form input:checkbox").wrap('<span></span>').parent().css({background:"yellow", border:"3px red solid"});
    $("div").text("O jQuery achou " + input.length + " elementos deste tipo.")
            .css("color", "red");
    $("form").submit(function () { return false; }); //  o evento submit não será invocado

</script>

</body>
</html>
