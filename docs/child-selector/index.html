Selector Child(“parent > child”)

Categorias: Selectors > Hierarchy

selector child

adicionado na versão: 1.0jQuery('parent > child')

  • parent
    Qualquer seletor válido.
    child
    Um seletor para filtrar os elementos filhos.

Descrição: Seleciona todos os elementos filhos diretor especificadors por elementos "child" de elementos especificador por "parent".

Como um seletor CSS, o combinador filho é suportado pela maioria dos browsers web modernos incluindo Safari, Firefox, Opera, Chrome e Internet Explorer 7 e acima, mais notavelmente não por Internet Explorer nas versões 6 e abaixo. Entretanto, em jQuery, este seletor (como todos os outros) trabalha com todos os browsers suportados, incluindo IE6.

O combinador filho (E > F) pode ser pensado como uma forma mais específica do combinador descendente (E F) no qual ele seleciona apenas os descendentes de primeiro nível.

Examplo:

Coloca uma borda em torno de todos os itens da lista que são filhos de <ul class="topnav"> .

<!DOCTYPE html>
<html>
<head>
  <style>
body { font-size:14px; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
                    
<ul class="topnav">
    <li>Item 1</li>
    <li>Item 2 
        <ul><li>Item aninhado 1</li><li>Item aninhado 2</li><li>Item aninhado 3</li></ul>
       </li>
    <li>Item 3</li>
</ul>

<script>$("ul.topnav > li").css("border", "3px double red");</script>

</body>
</html>
