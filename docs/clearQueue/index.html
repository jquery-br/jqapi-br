.clearQueue()

Categorias: Efeitos > Personalizado | Dados | Utilidades

.clearQueue( [ nome_da_fila ] ) Retornar para: jQuery

Descrição: Remove da fila todos os itens que ainda não foram executados.

  • adicionado na versão: 1.4 .clearQueue( [ nome_da_fila ] )

    nome_da_fila: Uma string contendo o nome da fila. O valor padrão é fx, esta é a fila padrão de efeitos que estão sendo executados.

Quando a função .clearQueue() é chamada, todas as funções que estão na fila e ainda não foram executadas são removidas. Quando utilizado sem um argumento, .clearQueue() remove as funções restantes da fila fx, a fila padrão de efeitos. É parecida com a função .stop(true). Entretanto, enquanto a função .stop() é utiliza somente para animações, .clearQueue() também pode ser utilizada para remover qualquer função que foi adicionada para uma fila jQuery genérica utilizando a função .queue().

Exemplo:

Esvazia a fila.

<!DOCTYPE html>
<html>
<head>
  <style>
div { margin:3px; width:40px; height:40px;
    position:absolute; left:0px; top:30px; 
    background:green; display:none; }
div.newcolor { background:blue; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="start">Start</button>
<button id="stop">Stop</button>
<div></div>
<script>$("#start").click(function () {
  $("div").show("slow");
  $("div").animate({left:'+=200'},5000);
  $("div").queue(function () {
    $(this).addClass("newcolor");
    $(this).dequeue();
  });
  $("div").animate({left:'-=200'},1500);
  $("div").queue(function () {
    $(this).removeClass("newcolor");
    $(this).dequeue();
  });
  $("div").slideUp();
});
$("#stop").click(function () {
  $("div").clearQueue();
  $("div").stop();
});</script>

</body>
</html>
