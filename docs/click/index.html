.click()

Categoria: Events > Mouse Events

.click( handler(eventObject) ) Retorna: jQuery

Descrição: Adiciona um manipulador de eventos ao evento "click" do JavaScript, ou disparar este evento em um elemento.

  • adicionado na versão: 1.0.click( handler(eventObject) )

    handler(eventObject)Função que será executada toda vez que o evento é disparado.

  • adicionado na versão: 1.4.3.click( [ eventData ], handler(eventObject) )

    eventDataMapa de dados a ser passado para o manipulador de eventos.

    handler(eventObject)Função que será executada toda vez que o evento é disparado

  • adicionado na versão: 1.0.click()

Este evento é um atalho para .bind('click', handler) nas primeiras duas variações, e .trigger('click') para a terceira.

O evento click é enviado a um elemento quando o ponteiro do mouse está sobre o elemento, e o botão do mouse é pressionado e liberado. Qualquer elemento HTML pode receber este evento.

Por exemplo, considere esse HTML:
<div id="target">
  Clique aqui
</div>
<div id="other">
  Acionar o manipulador
</div>

O manipulador de evento pode ser agregado a qualquer <div>:

$('#target').click(function() {
  alert('Manipulador para a chamada .click()');
});

Agora se clicarmos neste elemento, o alerta será apresentado:

Manipulador para a chamada .click()

Podemos também disparar o evento quando um diferente elemento é clicado:

$('#other').click(function() {
  $('#target').click();
});

Após este código executar, clique em Acionar o manipulador também irá apresentar a mensagem.

O evento click somente é disparado depois da exata série de eventos:

  • O botão do mouse é pressionado enquanto o ponteiro está dentro do elemento.
  • O botão do mouse é liberado enquanto o ponteiro está dentro do elemento.

Isso é geralmente a sequência desejada antes de realizar uma ação. Se isso não é obrigatório, o evento mousedown ou mouseuppode ser mais adequado.

Exemplo: Para ocultar parágrafos em uma página quando forem clicados: 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { color:red; margin:5px; cursor:pointer; }
  p.hilite { background:yellow; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Primeiro parágrafo</p>

  <p>Segundo parágrafo</p>
  <p>Mais um parágrafo</p>
<script>
    $("p").click(function () { 
      $(this).slideUp(); 
    });
    $("p").hover(function () {
      $(this).addClass("hilite");
    }, function () {
      $(this).removeClass("hilite");
    });
</script>

</body>
</html>
