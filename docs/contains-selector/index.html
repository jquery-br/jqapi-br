:contains() Selector

Categorias: Selectors > Content Filter

contains selector

Adicionado na versão: 1.1.4jQuery(':contains(text)')

  • text
    Uma string de texto para procurar. É case sensitive.

Descrição: Seleciona todos os elementos que contem o texto especificado.

O texto encontrado pode aparecer diretamente dentro do elemento selecionado, em qualquer descenente do elemento ou uma combinação deles. Como com seletores de valor de atributos, textos dentro dos parênteses de :contains() podem ser escritos como meras palavras ou envolvidas por aspas. O texto deve bater com o case (se é maiúsculo ou minúsculo) para ser selecionado.

Example:

Encontra todas das divs contendo "John" e as sublinha.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div>John Resig</div>

<div>George Martin</div>
<div>Malcom John Sinclair</div>
<div>J. Ohn</div>
      
    
<script>
$("div:contains('John')").css("text-decoration", "underline");
    </script>

</body>
</html>
