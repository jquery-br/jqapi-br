deferred.always()

Categories: Deferred Object

deferred.always( alwaysCallbacks ) Retorna: Deferred

Descrição: Adiciona manipuladores a serem chamados quando o objeto Deferred é tanto resolvido quanto rejeitado.

  • adicionado na versão: 1.6deferred.always( alwaysCallbacks )

    alwaysCallbacks Uma função ou array de funções, que é chamado quando o Deferred é tanto resolvido quanto rejeitado.

O argumento pode ser uma função simples ou uma array de funções. Quando o Deferred é resolvido ou rejeitado, o alwaysCallbacks é chamado. Quando deferred.always() retorna o objeto Deferred, outros métodos do objeto Deferred pode ser encadeado para este, incluindo adicionais métodos .always(). Quando o Deferred é resolvido ou rejeitado, os callbacks são executados na ordem em que eles são adicionados, usando os argumentos providos para chamadas de métodos resolve, reject, resolveWith ou rejectWith. Para mais informação, veja a documentaçãao para objeto Deferred.

Exemplo

Quando o método jQuery.get() retorna um objeto jqXHR, o qual é derivado de um objeto Deferred, podemos acoplar um callback para ambos sucesso e erro usando o método deferred.always().


$.get("test.php").always( function() {
  alert("$.get completo com argumentos de callback de sucesso ou erro");
} );
