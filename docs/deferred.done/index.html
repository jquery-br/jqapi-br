deferred.done()

Categories: Deferred Object

deferred.done( doneCallbacks, [ doneCallbacks ] ) Retorna: Deferred

Descrição: Adiciona manipuladores para serem chamados quando o objeto object é resolvido.

  • adicionado na versão: 1.5deferred.done( doneCallbacks, [ doneCallbacks ] )

    doneCallbacks Uma função ou uma array de funções que são chamadas quando Deferred é resolvido.

    doneCallbacks Funções ou array de funções adicionais opcionais que são chamadas quando Deferred é resolvido.

O método deferred.done() aceita um ou mais argumentos, todos dos quais podem ser tanto uma simples função ou uma array de funções. Quando Deferred é resolvido, o doneCallbacks é chamado. Callbacks são executados na ordem em que eles foram adicionados. Desde que deferred.done() retorna o objeto deferred, outros métodos do objeto deferred podem ser encadeados para ele, incluindo adicionais métodos .done(). Quando Deferred é resolvido, doneCallbacks são executadosusando os argumentos providos as chamadas de método resolve ou resolveWith na ordem em que eles foram adicionados. Para mais informação, veja a documentação para objeto Deferred.

Exemplos:

Exemplo: Quando o método jQuery.get retorna um objeto jqXHR, que é derivado de um objeto Deferred, podemos acoplar um callback de successo usando o .done() method. 


$.get("test.php").done(function() {
  alert("$.get sucedido");
});

Exemplo: Resolve um objeto Deferred quando o usuário clica em um botão, dispadando um númeto de funções callback: 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>

 <button>Go</button>
 <p>Ready...</p>

<script>
// 3 funções para chamar quando o objeto Deferred é resolvido
function fn1() {
  $("p").append(" 1 ");
}
function fn2() {
  $("p").append(" 2 ");
}
function fn3(n) {
  $("p").append(n + " 3 " + n);
}

// cria um objeto deferred
var dfd = $.Deferred();

// adiciona manipuladores para serem chamados quando dfd é resolvido
dfd
// .done() pode levar qualquer número de funções ou array de funções
.done( [fn1, fn2], fn3, [fn2, fn1] )
// podemos encadear os métodos feitos, também
.done(function(n) {
  $("p").append(n + " feito.");
});

// resolve o objeto Deferred quando o botão é clicado
$("button").bind("click", function() {
  dfd.resolve("and");
});
</script>

</body>
</html>
