deferred.fail()

Categories: Deferred Object

deferred.fail( failCallbacks, [ failCallbacks ] ) Retorna: Deferred

Descrição: Adiciona manipuladores para serem chamados quando o objeto Deferred é rejeitado.

  • adicionado na versão: 1.5deferred.fail( failCallbacks, [ failCallbacks ] )

    failCallbacks Uma função, uma função ou uma array de funções que são chamadas quando o Deferred é rejeitado.

    failCallbacks Funções ou array de funções opcionais adicionais que são chamadas quando o é rejeitado.

O método deferred.fail() aceita um ou mais argumentos, todos os quais podem ser ou ser uma função simples ou uma array de funções. Quando o Deferred é rejeitado, os failCallbacks são chamados. Callbacks são executados na ordem em que eles são adicionados. Quando deferred.fail() retorna o objeto deferred, outros métodos do objoeto deferred podem ser encadeados para ele, incluindo adicionais métodos deferred.fail(). Os failCallbacks são executados usando os argumentos providos para as chamadas de métodos deferred.reject() ou deferred.rejectWith() na ordem em que eles foram adicionados. Para mais informação, veja a documentação para objeto Deferred.

Exemplo:

Quando o método jQuery.get retorna um objeto jqXHR, que é derivado de um Deferred, você pode acoplar um callback de sucesso e falha usando os métodos deferred.done() e deferred.fail().


$.get("test.php")
  .done(function(){ alert("$.get succeeded"); })
  .fail(function(){ alert("$.get failed!"); });
