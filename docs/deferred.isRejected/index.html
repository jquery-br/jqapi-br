deferred.isRejected()

Categories: Deferred Object

deferred.isRejected() Retorna: Boolean

Descrição: Determina se um objeto Deferred foi rejeitado.

  • adicionado na versão: 1.5deferred.isRejected()

Retorna true se o objeto Deferred está no estado rejeitado, significando que ambos deferred.reject() ou deferred.rejectWith() tem sido chamados para o objeto e os failCallbacks foram chamados (ou estão no processo de serem chamados).

Note que um objeto Deferred pode estar em um dos três estados: não resolvido, resolvido ou rejeitado; use deferred.isResolved() para dererminar se o objeto Deferred está no estado resolvido. Esses métodos são primordialmente úteis para debug, por exemplo, para determinar se um Deferred já foi resolvido mesmo quando você estiver dentro do código que tentou rejeitar.
