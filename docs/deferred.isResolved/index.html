deferred.isResolved()

Categories: Deferred Object

deferred.isResolved() Retorna: Boolean

Descrição: Determina se um objeto Deferred foi resolvido.

  • adicionado na versão: 1.5deferred.isResolved()

Retorna true se o objeto Deferred está no estado resolvido, significando que tanto deferred.resolve() quanto deferred.resolveWith() foram chamados para o objeto e os doneCallbacks foram chamados (ou estão no processo para serem chamados).

Note que um objeto Deferred pode estar em um dos três estados: não-resolvido, resolvido ou rejeitado; use deferred.isRejected() para determinar se o objeto Deferred está no estado rejeitado. Esses métodos são primordialmente úteis para debugar, por exemplo para determinar se um Deferred já foi resolvido mesmo se você estiver dentro do código que pretendei rejeitá-lo.
