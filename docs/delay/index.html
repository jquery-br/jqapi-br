.delay()

Categorias: Efeitos > Personalizado

.delay( duração, [ nome_da_fila ] ) Retornar para: jQuery

Descrição: Configura um tempo de atraso na execução do itens subsequentes na fila.

  • versão adicionada: 1.4 .delay( duração, [ nome_da_fila ] )

    duração: Um valor inteiro indicando o número em millisegundos para atrasar a execução do próximo item que está na fila.

    nome_da_fila: Uma string contendo o nome da fila. O valor padrão é fx, esta é a fila padrão de efeitos que estão sendo executados.

Adicionado ao jQuery na versão 1.4, a função .delay() nos permite atrasar a execução das funções que seguem na fila. Pode ser utilizada com a fila de efeitos padrão ou com uma fila personalizada. Somente eventos posteriores que estão na fila são atrasados; por exemplo, isto não atrasará os formulários sem argumentos de .show() ou .hide() os quais não utilizam a fila de efeitos.

A duração é passada através do valor em milisegundos; valores maiores indicam animações mais lentas e não mais rápidas. Os valores 'fast' e 'slow' podem ser fornecidos para indicar a duração de 200 e 600 milisegundos respectivamente.

Utilizando a fila de efeitos padrão, nós podemos por exemplo, adicionar um atraso de 800 milisegundos entre .slideUp() e .fadeIn() do elemento <div id="foo">:

$('#foo').slideUp(300).delay(800).fadeIn(400);

Quando esta função é executada, o elemento realiza o efeito de slide (escorregar) para cima por 300 milisegundos e pausa por 800 milisegundos antes de aplicar o efeito fade(sumir) em 400 milisegundos.

A função .delay() foi feita para atrasar efeitos jQuery que estão na fila. Mas, por que é limitado— por exemplo, não oferece uma maneira de cancelar o atraso— a função .delay() não é um substituto para a função JavaScript nativa setTimeout , a qual é mais apropriada para certos casos.

exemplo:

Anima escondendo e mostrando os dois elementos DIVs, atrasando antes de mostrar a primeira DIV novamente.

<!DOCTYPE html>
<html>
<head>
  <style>
div { width: 60px; height: 60px; float: left; }
.primeira { background-color: #3f3; }
.segunda { background-color: #33f;}
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<p><button>Run</button></p>
<div class="primeira"></div>
<div class="segunda"></div>
	
<script>
    $("button").click(function() {
      $("div.primeira").slideUp(300).delay(800).fadeIn(400);
      $("div.segunda").slideUp(300).fadeIn(400);
    });
</script>

</body>
</html>
