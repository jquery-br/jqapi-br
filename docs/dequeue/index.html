.dequeue()

Categoria: Efeitos > Personalizado | Dados | Utilidades

.dequeue( [ nome_da_fila ] ) Retornar para: jQuery

Descrição: Executa a próxima função que esta na fila através dos elementos encontrados.

  • adicionado na versão: 1.2 .dequeue( [ nome_da_fila ] )

    nome_da_fila: Uma string contendo o nome da fila. O valor padrão é fx, esta é a fila padrão de efeitos que estão sendo executados.

Quando a função .dequeue() é chamada, a próxima função que esta na fila é removida da fila e é executada. Esta função deve (diretamente ou indiretamente) chamar a função .dequeue() para que a sequência da fila possa continuar.

Exemplo:

Utiliza a função dequeue para encerrar uma função que esta na fila permitindo com que a fila continue sua sequência.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:3px; width:50px; position:absolute;
  height:50px; left:10px; top:30px; 
  background-color:yellow; }
  div.red { background-color:red; }  
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Start</button>  
<div></div>
<script>
$("button").click(function () {
  $("div").animate({left:'+=200px'}, 2000);
  $("div").animate({top:'0px'}, 600);
  $("div").queue(function () {
    $(this).toggleClass("red");
    $(this).dequeue();
  });
  $("div").animate({left:'10px', top:'30px'}, 700);
});
</script>

</body>
</html>
