.dequeue()

Categorias: Effects > Custom | Data | Utilities

.dequeue( [ queueName ] ) Retorna: jQuery

Descrição: Executa a próxima função da queue para um grupo de elementos

  • adicionado na versão: 1.2.dequeue( [ queueName ] )

    queueNameUma string contendo o nome da queue. Padrão fx, o padrao para queue de efeitos.

Quando .dequeue() é chamado, a próxima função da queue é removida e então executada. Essa função por sua vez, deve (direta ou indiretamente) causar a chamada da .dequeue(), então a sequencia pode continuar.

Exemplo:

Use dequeue para finalizar uma função queue personalizada que permite a queue continuar.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:3px; width:50px; position:absolute;
  height:50px; left:10px; top:30px; 
  background-color:yellow; }
  div.red { background-color:red; }  
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Start</button>  
<div></div>
<script>
$("button").click(function () {
  $("div").animate({left:'+=200px'}, 2000);
  $("div").animate({top:'0px'}, 600);
  $("div").queue(function () {
    $(this).toggleClass("red");
    $(this).dequeue();
  });
  $("div").animate({left:'10px', top:'30px'}, 700);
});
</script>

</body>
</html>
