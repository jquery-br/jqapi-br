Descendant Selector (“ancestor descendant”)

Categorias: Selectors > Hierarchy

descendant selector

adicionado na versão: 1.0jQuery('ancestor descendant')

  • ancestor
    Qualquer seletor válido.
    descendant
    Um seletor para filtrar os elementos descendentes.

Descrição: Seleciona todos elementos que são descendentes de um determinado ancestral.

Um descendente de um elemento poderia ser um filho, neto, bisneto, etc, desse elemento.

Exemplo:

Encontra todos input descendentes do form.

<!DOCTYPE html>
<html>
<head>
  <style>

  body { font-size:14px; }
  form { border:2px green solid; padding:2px; margin:0; 
         background:#efe; }
  div { color:red; }
  fieldset { margin:1px; padding:3px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>
    <div>Form is surrounded by the green outline</div>
    <label>Child:</label>
    <input name="name" />

    <fieldset>
      <label>Grandchild:</label>
      <input name="newsletter" />
    </fieldset>

  </form>
  Sibling to form: <input name="none" />
<script>$("form input").css("border", "2px dotted blue");</script>

</body>
</html>
