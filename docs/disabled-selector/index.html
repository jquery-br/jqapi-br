Categorias: Selectors > Form

seletor disabled

adicionado na versão: 1.0jQuery(':disabled')

Descrição: Seleciona todos os elementos que estãao desabilitados.

Como em seletores de pseudo-classes (aqueles que começam por um ":") é recomendado precedê-lo com uma tag name ou algum outro seletor; senão o seletor ("*") é implicado. Em outras palavras, o mero $(':disabled') é equivalente a $('*:disabled'), então $('input:disabled') deveria ser usado ao invés disso.

Exemplo:

Encontra todos os elementos input que estão desabilitados.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>

    <input name="email" disabled="disabled" />
    <input name="id" />
  </form>
<script>$("input:disabled").val("this is it");</script>

</body>
</html>
