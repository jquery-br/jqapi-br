Element Selector (“element”)

Categorias: Selectors > Basic

element selector

adicionado na versão: 1.0jQuery('element')

  • element
    Um elemento a ser buscado. Refere-se ao tagName do nó do DOM.

Descrição: Seleciona todos os elementos com a tagName dado.

A função getElementsByTagName() do JavaScript é chamada para retornar os elementos apropriados quando essa expressão é usada.

Exemplo:

Encontra todos os elementos DIV.

<!DOCTYPE html>
<html>
<head>
  <style>
  div,span {
    width: 60px;
    height: 60px;
    float:left;
    padding: 10px;
    margin: 10px;
    background-color: #EEEEEE;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>DIV1</div>

  <div>DIV2</div>
  <span>SPAN</span>
<script>$("div").css("border","9px solid red");</script>

</body>
</html>
