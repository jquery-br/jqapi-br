:empty Selector

Categorias: Selectors > Content Filter

empty selector

adicionado na versão: 1.0jQuery(':empty')

Descrição: Seleciona todos os elementos que não tem nenhum filho (incluindo nós de texto).

É o inverso do :parent.

Uma coisa importante de notar, :empty (e :parent) incluem como elementos filho nós de texto.

A W3C recomenda que o elemento <p> tenha ao menos um nó filho, mesmo esse filho sendo um mero texto (veja http://www.w3.org/TR/html401/struct/text.html#edef-P). Por outro lado, alguns outros elementos são vazios (i.e não tem filhos) por definição: <input>, <img>, <br>, e <hr>, como exemplo.

Exemplo:

Encontra todos os elementos vazios - Eles não tem elementos filhos ou texto.

<!DOCTYPE html>
<html>
<head>
  <style>

  td { text-align:center; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <table border="1">
    <tr><td>TD #0</td><td></td></tr>
    <tr><td>TD #2</td><td></td></tr>

    <tr><td></td><td>TD#5</td></tr>
  </table>
<script>$("td:empty").text("Estava vazio!").css('background', 'rgb(255,220,200)');</script>

</body>
</html>
