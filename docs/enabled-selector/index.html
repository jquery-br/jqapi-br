:enabled Selector

Categorias: Selectors > Form

enabled selector

adicionado na versão: 1.0jQuery(':enabled')

Descrição: Seleciona todos os elementos que estão habilitados.

Assim como outros seletores pseudo-class (os que começam com um ":") é recomendado ser precedido por alguma tag ou qualquer outro seletor; caso contrário, o seletor universal ("*") é aplicado. Em outra palavras, o $(':enabled') é equivalente a $('*:enabled'), então $('input:enabled') deve ser usado ao invés.

Exemplo:

Encontra todos os elementos input que estão habilitados.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>

    <input name="email" disabled="disabled" />
    <input name="id" />
  </form>
<script>$("input:enabled").val("É esse");</script>

</body>
</html>
