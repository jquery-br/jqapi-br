Seletor :eq()

Categorias: Selectors > Basic Filter

Seletor eq

adicionado na versão: 1.0jQuery(':eq(index)')

  • index
    Baseado no index 0.

Descrição: Seleciona o elemento no index n.

Os seletores relacionados com o index (:eq(), :lt(), :gt(), :even, :odd) filtram os elementos que correspondem as expressões que os procedem. Elas limitam a busca do elemento para baixo de sua linha na DOM. Por exemplo, se elementos são primeiramente selecionados com o seletor (.myclass) e quatro elementos são retornados, esses elementos terão os indices 0 até 3 devido a esse seletor.

Note, desde que os arrays do JavaScript são baseados no index 0, esses seletores refletem esse fato. É por isso que $('.myclass:eq(1)') seleciona o segundo elemento no documento com a classe myclass, ao invés do primeiro. Em contrate, :nth-child(n) tem o index baseado em 1 em conformidade com as especificações da CSS.

Diferente do método .eq(index), o seletor :eq(index) não aceita um valor negativo para o index. Exemplo, enquanto $('li').eq(-1) seleciona o ultimo elemento li, $('li:eq(-1)') não seleciona nada.

Exemplo: Procura o terceiro td. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <table border="1">
  <tr><td>TD #0</td><td>TD #1</td><td>TD #2</td></tr>
  <tr><td>TD #3</td><td>TD #4</td><td>TD #5</td></tr>
  <tr><td>TD #6</td><td>TD #7</td><td>TD #8</td></tr>
</table>
<script>$("td:eq(2)").css("color", "red");</script>

</body>

Exemplo: Aplica tres estilos diferente na lista de items para demonstrar que o :eq() é projetado para selecionar apenas um elemento enquanto o :nth-child() ou :eq() dentro de um construtor de loop como .each() pode selecionar vários elementos. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <ul class="nav">
   <li>Lista 1, item 1</li>
   <li>Lista 1, item 2</li>
   <li>Lista 1, item 3</li>
</ul>
<ul class="nav">
  <li>Lista 2, item 1</li>
  <li>Lista 2, item 2</li>
  <li>Lista 2, item 3</li>
</ul>

<script>
// aplica fundo amarelo para apenas um <li>
$("ul.nav li:eq(1)").css( "backgroundColor", "#ff0" );

// aplica italico no segundo <li> dentro de cada <ul class="nav">
$("ul.nav").each(function(index) {
  $(this).find("li:eq(1)").css( "fontStyle", "italic" );
});

// aplica cor vermelha nos descendentes do <ul class="nav">
// para cada <li> que é o segundo filho
$("ul.nav li:nth-child(2)").css( "color", "red" );
</script>

</body>
</html>
