Seletor :even

Categorias: Selectors > Basic Filter

Seletor even

adicionado na versão: 1.0jQuery(':even')

Descrição: Seleciona os elementos par, baseado no index 0, Veja também odd.

Em particular, note que baseado no inxex 0 significa, que o:even seleciona o primeiro elemento, terceiro e assim por diante.

Exemplo:

Procura por linhas pares na tabela, marcando o primeiro, terceiro e assim por diante (index 0, 2, 4 etc.).

<!DOCTYPE html>
<html>
<head>
  <style>

  table {
    background:#eeeeee;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <table border="1">
    <tr><td>Linha com index #0</td></tr>
    <tr><td>Linha com index #1</td></tr>

    <tr><td>Linha com index #2</td></tr>
    <tr><td>Linha com index #3</td></tr>
  </table>
<script>$("tr:even").css("background-color", "#bbbbff");</script>

</body>
</html>
