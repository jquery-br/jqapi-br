event.timeStamp

Categorias: Events > Event Object

event.timeStamp Retorna: Number

Descrição: A diferença em milisegundos entre a hora que o evento é disparado e 1 de janeiro de 1970.

  • adicionado na versão: 1.2.6event.timeStamp

Essa propriedade pode ser útil para medições de performance de certas funções do jquery, egando o valor do event.timeStamp em diferentes pontos do código e notando a diferença.

Exemplo:

Mostra o tempo desde o ultimo clique.

<!DOCTYPE html>
<html>
<head>
  <style>
div { height: 100px; width: 300px; margin: 10px; 
      background-color: #ffd; overflow: auto; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div>Click.</div>
<script>
var last, diff;
$('div').click(function(event) {
  if ( last ) {
    diff = event.timeStamp - last
    $('div').append('Tempo desde o ultimo evento: ' + diff + '<br/>');
  } else {
    $('div').append('Click novamente.<br/>');
  }
  last = event.timeStamp;
});  
</script>

</body>
</html>
