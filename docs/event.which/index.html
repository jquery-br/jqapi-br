event.which

Categories: Events > Event Object

event.which Retorna: Number

Descrição: Para eventos em botões ou teclas, esse atributo indica qual botão ou tecla foram apertados.

  • Adicionado na versão: 1.1.3event.which

O event.which normaliza event.keyCode e event.charCode. É recomendo o uso do event.which para entradas através do teclado. Leia mais em event.charCode no MDC.

Example:

Registro de qual tecla foi apertado.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<input id="whichkey" value="type something">
<div id="log"></div>
<script>$('#whichkey').bind('keydown',function(e){ 
  $('#log').html(e.type + ': ' +  e.which );
});  </script>

</body>
</html>
