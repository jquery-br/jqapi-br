.fadeToggle()

Categories: Effects | Effects > Fading

.fadeToggle( [ duration ], [ easing ], [ callback ] ) Retorna: jQuery

Descrição: Mostra ou esconde os elementos encontrados animando sua opacidade.

  • Adicionado na versão: 1.4.4.fadeToggle( [ duration ], [ easing ], [ callback ] )

    durationUma palavra ou número que determina por quanto tempo durará a animação.

    easingUma palavra que determina qual função de suavização será usada para a transição.

    callbackUma função para ser executada quando a animação finalizar.

O método .fadeToggle() anima a opacidade dos elementos encontrados. Quando um elemento visível é chamado, a propriedade de estilo display é definida como none quando a opacidade chegar a 0, então o elemento não aparecerá no layout da página.

A duração é explicita em milisegundos; Valores maiores indicam animações mais demoradas, não mais rápidas. As palavras 'fast' e 'slow' podem ser fornecidas para indicar a duração de 200 e 600 milisegundos, respectivamente.

Funções easing(suavização) especificam a velocidade que a animação transcorrerá nos disferentes pontos da animação. Há apenas implementações padrões na biblioteca jQuery, uma chamada swing, e outra que progride em um ritmo constante, chamada linear. Mais funções de easing estão disponíveis com o uso de plug-ins, o mais conhecido é jQuery UI suite.

Se fornecido, o callback é disparado uma vez que a animação tenha finalizado. Isto pode ser útil para amarrar animações diferentes juntas em sequência. O callback não envia qualquer argumento, mas this é definido apenas para elementos DOM serem animados. Se vários elementos são animados, é importante observar que o callback é executado para cada elemento correspondente, não apenas para a animação como um todo.

Observação adicional:

  • Todos efeitos jQuery, incluindo .fadeToggle(), podem ser desligados globalmente pela configuração jQuery.fx.off = true, que efetivamente define a duration para 0. Para mais informações, consulte jQuery.fx.off.

Exemplo:

Esconde ou mostra o primeiro parágrafo gradualmente, completando a animação dentro de 600 milisegundos e usando uma suavização linear. Esconde ou mostra o último parágrafo gradualmente dentro de 200 milisegundos, adicionando a mensagem "finished" quando estiver finalizado. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<button>fadeToggle p1</button>
<button>fadeToggle p2</button>
<p>This paragraph has a slow, linear fade.</p>

<p>This paragraph has a fast animation.</p>
<div id="log"></div>

<script>
$("button:first").click(function() {
  $("p:first").fadeToggle("slow", "linear");
});
$("button:last").click(function () {
  $("p:last").fadeToggle("fast", function () {
    $("#log").append("<div>finished</div>");
  });
});
</script>

</body>
</html>
