.find()

Categorias: Traversing > Tree Traversal

.find( selector ) Retorna: jQuery

Descrição: Obtém os descendentes de cada elemento do conjunto atual de elementos encontrados, filtrados por um selector, objeto jQuery ou elemento.

  • adicionado na versão: 1.0 .find( selector )

    selector String contendo uma expressão de seleção para filtrar os elementos do conjunto.

  • adicionado na versão: 1.6 .find( jQuery object )

    jQuery object Objeto jQuery para filtrar os elementos do conjunto.

  • adicionado na versão: 1.6 .find( element )

    element Elemento para filtrar os elementos do conjunto.

Dado um objeto jQuery que representa um conjunto de elementos DOM, o método .find() permite a busca através dos descendentes desses elementos na árvore DOM e constrói um novo objeto jQuery combinando os elementos resultantes. Os métodos .find() e .children() são similares, exceto pelo último percorrer em um único nível para baixo na árvore DOM.

A primeira assinatura do método .find() aceita uma expressão de seleção (selector) do mesmo tipo que pode ser aplicado a função $(). Os elementos serão filtrados através de testes de correspondência com esse selector.

Considere a seguinte página com uma lista básica aninhada:

<ul class="level-1">
  <li class="item-i">I</li>
  <li class="item-ii">II
    <ul class="level-2">
      <li class="item-a">A</li>
      <li class="item-b">B
        <ul class="level-3">
          <li class="item-1">1</li>
          <li class="item-2">2</li>
          <li class="item-3">3</li>
        </ul>
      </li>
      <li class="item-c">C</li>
    </ul>
  </li>
  <li class="item-iii">III</li>
</ul>

Se começarmos com o item II, podemos encontrar os itens da lista dentro dele:

$('li.item-ii').find('li').css('background-color', 'red');

O resultado dessa chamada é um fundo vermelho para os itens A, B, 1, 2, 3 e C. Embora o item II corresponda a expressão de seleção (selector), ele não é incluído no resultado; apenas descendentes são considerados candidatos para busca.

Ao contrário dos outros métodos de percorrer a árvore, a expressão de seleção (selector) é requirida na chamada de .find(). Se precisarmos recuperar todos os elementos descendentes podemos passar o selector universal '*' para fazer isso.

O selector contexto é implementado com o método .find(); portanto, $('li.item-ii').find('li') é equivalente a $('li', 'li.item-ii').

A partir do jQuery 1.6 também é possível filtrar a seleção através de uma coleção jQuery ou elemento. Com a mesma lista aninhada acima, se começarmos com:

var $allListElements = $('li');

E então passar esse objeto jQuery para find:

$('li.item-ii').find( $allListElements );

Retornará uma coleção jQuery que contém apenas a lista de elementos que são descendentes do item II.

Da mesma forma, um elemento também pode ser passado para find:

var item1 = $('li.item-1')[0];
$('li.item-ii').find( item1 ).css('background-color', 'red');

O resultado dessa chamada seria um fundo vermelho no item 1

Exemplos:

Exemplo: Selecionar todos parágrafos e buscar por elementos span descendentes, similar a $("p span") 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p><span>Olá</span>, como vai?</p>
<p>Eu? Estou <span>bem</span>.</p>
<script>
  $("p").find("span").css('color','red');
</script>

</body>
</html>

Exemplo: Obter uma coleção jQuery com todas as tags span. Spans dentro de tags p são alteradas para vermelho, o restante é deixado em azul. 

<!DOCTYPE html>
<html>
<head>
  <style>
    span { color: blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p><span>Olá</span>, como vai?</p>
  <p>Eu? Estou <span>bem</span>.</p>
  <div>Você <span>já</span> comeu?</div>
<script>
  var $spans = $('span');
  $("p").find( $spans ).css('color','red');
</script>

</body>
</html>

Demonstração:

Exemplo: Adiciona tags spans em torno de cada palavra e em seguida adiciona um hover que troca a cor de fundo da palavavra quando o ponteiro do mouse passa por cima dela. Também coloca itálico nas palavras que contém a letra t 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { font-size:20px; width:200px; cursor:default; 
      color:blue; font-weight:bold; margin:0 10px; }
  .hilite { background:yellow; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>
  Quando o dia é curto
  descubra o que é importante para você
  ou pare de acreditar
  </p>
<script>
  var newText = $("p").text().split(" ").join("</span> <span>");
  newText = "<span>" + newText + "</span>";

  $("p").html( newText )
    .find('span')
    .hover(function() { 
      $(this).addClass("hilite"); 
    },
      function() { $(this).removeClass("hilite"); 
    })
  .end()
    .find(":contains('t')")
    .css({"font-style":"italic", "font-weight":"bolder"});

</script>

</body>
</html>
