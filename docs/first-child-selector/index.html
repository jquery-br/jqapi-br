Seletor :first-child

Categorias: Seletores > Filtro de filhos

Seletor first-child

adicionado na versão: 1.1.4jQuery(':first-child')

Descrição: Seleciona todos os elementos que são o primeiro filho de seus pais.

Enquanto :first seleciona apenas um elemento, o seletor:first-child pode selecionar mais que um: um para cada pai. Isso é equivalente ao :nth-child(1).

Exemplo:

Encontra o primeiro span em cada div selecionada.

<!DOCTYPE html>
<html>
<head>
  <style>
  span { color:#008; }
  span.sogreen { color:green; font-weight: bolder; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>
    <span>John,</span>
    <span>Karl,</span>
    <span>Brandon</span>

  </div>
  <div>
    <span>Glen,</span>
    <span>Tane,</span>
    <span>Ralph</span>

  </div>
<script>
    $("div span:first-child")
        .css("text-decoration", "underline")
        .hover(function () {
              $(this).addClass("sogreen");
            }, function () {
              $(this).removeClass("sogreen");
            });

</script>

</body>
</html>
