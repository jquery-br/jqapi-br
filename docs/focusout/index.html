.focusout()

Categories: Events > Keyboard Events | Events > Mouse Events

.focusout( handler(eventObject) ) Retorna: jQuery

Descrição: Trata um manipulador de evento para o evento Javascript "focusout".

  • adicionado na versão: 1.4.focusout( handler(eventObject) )

    handler(eventObject)Uma função para executar a cada vez que o evento é disparado.

  • adicionado na versão: 1.4.3.focusout( [ eventData ], handler(eventObject) )

    eventDataUm mapa de dados que passará ao manipulador de evento.

    handler(eventObject)Uma função para executar a cada vez que o evento é disparado.

Este método é um atalho para .bind('focusout', handler).

O evento focusout é enviado a um elemento quando ele, ou algum elemento dentro dele, perder o foco. Isto é diferente do evento blur na qual ele suporta a detecção da perda de foco dos elementos pais (em outras palavras, ele suporta a bolha de eventos).

Este evento será como usado junto ao evento focusin.

Exemplo:

Observe uma perda de foco para ocorrer dentro dos parágrafos e note a diferença entre o contador focusout e o blur.

<!DOCTYPE html>
<html>
<head>
  <style>
.inputs { float: left; margin-right: 1em; }
.inputs p { margin-top: 0; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div class="inputs">
  <p>
    <input type="text" /><br />
    <input type="text" /> 
  </p>
  <p>
    <input type="password" />
  </p>
</div>
<div id="fo">focusout fire</div>
<div id="b">blur fire</div>

<script>
var fo = 0, b = 0;
$("p").focusout(function() {
  fo++;
  $("#fo")
    .text("focusout fired: " + fo + "x");
}).blur(function() {
  b++;
  $("#b")
    .text("blur fired: " + b + "x");
  
});
</script>

</body>
</html>
