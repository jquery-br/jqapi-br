.get()

Categories: Miscellaneous > DOM Element Methods

.get( [ index ] ) Retorna: Element, Array

Description: Obtém os elementos DOM encontrados pelo objeto jQuery.

  • versão incluída: 1.0.get( [ index ] )

    indexUm valor inteiro baseado no zedo indicando qual elemento obter.

O método .get() nos garante acesso aos nós DOM sob cada objeto jQuery. Suponha que nós tenhamos uma simples lista não-ordenada na página:

<ul>
  <li id="foo">foo</li>
  <li id="bar">bar</li>
</ul>

Sem um parâmetro, .get() retorna todos os elementos:

alert($('li').get());

Todos os nós DOM retornados por esta chamada, contidas numa array padrão:

[<li id="foo">, <li id="bar">]

Com um índice especificado, .get() vai obter um único elemento:

($('li').get(0));

Desde que o índice seja baseado em zero, o primeiro elemento da lista é retornado:

<li id="foo">

Cada objeto jQuery também é mascarado como uma array, então podemos usar o operador de deferência para pegar, ao invés disso, ao menos um item da list:

alert($('li')[0]);

Entretanto, esta intaxe falta uma das capacitades adicionais de .get(), como o índice de especificação negativa:

alert($('li').get(-1));

Um índice negativo é contato do final da série encontrada, assim este exemplo retornará o último item na lista:

<li id="bar">

Exemplo: Seleciona todas as divs no documento e retorna os Elementos DOM como uma Array, então usa o método-reverso embutido para reverter essa array. 

<!DOCTYPE html>
<html>
<head>
  <style>
  span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Reversed - <span></span>

  <div>One</div>
  <div>Two</div>
  <div>Three</div>
<script>

    function disp(divs) {
      var a = [];
      for (var i = 0; i < divs.length; i++) {
        a.push(divs[i].innerHTML);
      }
      $("span").text(a.join(" "));
    }
    
    disp( $("div").get().reverse() );
</script>

</body>
</html>

Exemplo: Dá o nome da tag do elemento clicado. 

<!DOCTYPE html>
<html>
<head>
  <style>
  span { color:red; }
  div { background:yellow; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <span>&nbsp;</span>
  <p>Neste parágrafo está umm <span>importante</span> seleção</p>

  <div><input type="text" /></div>
<script>

    $("*", document.body).click(function (e) {
      e.stopPropagation();
      var domEl = $(this).get(0);
      $("span:first").text("Clicado em - " + domEl.tagName);
    });
</script>

</body>
</html>
