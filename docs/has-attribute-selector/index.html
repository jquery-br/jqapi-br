Seletor atributo Has [nome]

Categorias: Seletores > Atributos

Seletor attributeHas

adicionado na versão: 1.0jQuery('[attribute]')

  • atributo
    O nome de um atributo.

Descrição: Seleciona elementos que tem o atributo especificado, com qualquer valor.

Exemplo:

Vincula um click que adiciona o id da div no texto.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>sem id</div>
  <div id="hey">com id</div>

  <div id="there">tem um id</div>
  <div>não</div>
<script>

    $('div[id]').one('click', function(){
      var idString = $(this).text() + ' = ' + $(this).attr('id');
      $(this).text(idString);
    });
</script>

</body>
</html>
