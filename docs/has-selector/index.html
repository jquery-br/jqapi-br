Categorias: Seletores > Filtro de conteúdo

Seletor has

adicionado na versão: 1.1.4jQuery(':has(seletor)')

  • seletor
    Qualquer seletor.

Descrição: Seleciona elementos que contenha no mínimo um elemento que bate com o seletor especificado.

A expressão $('div:has(p)') seleciona <div> se um <p> existir em qualquer lugar como descendente, não apenas como filho direto.

Exemplo:

Adiciona a classe "test" em todas as divs que tem um paragrafo dentro.

<!DOCTYPE html>
<html>
<head>
  <style>
  .test{ border: 3px inset red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div><p>Olá em um paragrafo</p></div>

  <div>Olá denovo! (sem paragrafo)</div>
<script>$("div:has(p)").addClass("test");</script>

</body>
</html>
