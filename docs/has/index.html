.has()

Categorias: Traversing > Filtering

.has( selector ) Retorna: jQuery

Descrição: Reduz o conjunto de elementos encontrados para um novo conjunto em que seus elementos possuem descendentes que coincidem com o selector ou elemento DOM.

  • adicionado na versão: 1.4.has( selector )

    selectorString contendo uma expressão de seleção para filtrar os elementos do conjunto..

  • adicionado na versão: 1.4.has( contained )

    containedElemento DOM para filtrar os elementos do conjunto.

Dado um objeto jQuery que representa um conjunto de elementos DOM, o método .has() constrói um novo objeto jQuery com um sub-conjunto dos elementos pesquisados. O selector fornecido é testado nos descendentes dos elementos pesquisados; o elemento será incluído no resultado se algum dos seus elementos descendentes combinar com o selector.

Considere a seguinte página com uma lista básica aninhada:

 <ul>
  <li>item da lista 1</li>
  <li>item da lista 2
    <ul>
      <li>item da lista 2-a</li>
      <li>item da lista 2-b</li>
    </ul>
  </li>
  <li>item da lista 3</li>
  <li>item da lista 4</li>
</ul>

Podemos aplicar esse método para o conjunto de itens a seguir:

$('li').has('ul').css('background-color', 'red');

O resultado dessa chamada aplica um fundo vermelho para o item 2, pois ele é o único <li> que possui um <ul> em seus descendentes.

Exemplo:

Verificar se um elemento está dentro de outro.

<!DOCTYPE html>
<html>
<head>
  <style>
  .full { border: 1px solid red; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<ul><li>Será que a UL contém uma LI?</li></ul>

<script>
  $("ul").append("<li>" + ($("ul").has("li").length ? "Sim" : "Não") + "</li>");
  $("ul").has("li").addClass("full");
</script>

</body>
</html>
