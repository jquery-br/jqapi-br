Seletor :hidden

Categorias: Seletores > Filtro de visibilidade

Seletor hidden

adicionado na versão: 1.0jQuery(':hidden')

Descrição: Seleciona todos os elementos que estão escondidos.

Um elemento pode ser considerado escondido por várias razões:

  • Eles têm a propriedade do css display como none.
  • São elementos de formulário do comtype="hidden".
  • Sua altura e largura são explicitamente setados em 0.
  • Um elemento antecessor é oculto, de modo que o elemento nõ é mostrado na página.

Elementos com visibility: hidden ou opacity: 0 são considerados visíveis, uma vez que eles continuam ocupando espaço no layout. Durante animações que esconde um elemento, o elemento é considerado visível até que a animação termine. Durante animações para mostrar um elemento, o elemento é considerado visível no início da animação.

Como o :hidden é determinado foi mudado no jQuery 1.3.2. Um elemento que se presume ser escondido se esse ou qualquer um de seus pais não consume nenhum espaço no documento. A propriedade do CSS visibility não entra na conta (portanto $(elem).css('visibility','hidden').is(':hidden') == false). Para mais detalhes acesse notas da release.

Exemplo:

Mostra todos as divs escondidas e conta os inputs escondidos.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width:70px; height:40px; background:#ee77ff; margin:5px; float:left; }
  span { display:block; clear:left; color:red; }
  .starthidden { display:none; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <span></span>
  <div></div>
  <div style="display:none;">Hider!</div>
  <div></div>

  <div class="starthidden">Hider!</div>
  <div></div>
  <form>
    <input type="hidden" />

    <input type="hidden" />
    <input type="hidden" />
  </form>
  <span>

  </span>
<script>
    // em alguns navegadores :hidden incluem head, title, script, etc... então limite ao body
    $("span:first").text("Encontrou no total " + $(":hidden", document.body).length + 
                         " elementos escondidos.");
    $("div:hidden").show(3000);
    $("span:last").text("Encontrou " + $("input:hidden").length + " inputs escondidos."); 
</script>

</body>
</html>
