.hide()

Categories: Effects > Basics

.hide( ) Retorna: jQuery

Descrição: Oculta o elemento alvo do seletor jQuery.

  • criado na versão: 1.0.hide()

  • criado na versãod: 1.0.hide( duration, [ callback ] )

    durationUm texto ou número determinando o tempo de duração da animação.

    callbackFunção a ser chamada ao finalizar a animação.

  • criado na versão: 1.4.3.hide( [ duration ], [ easing ], [ callback ] )

    durationUm texto ou número determinando o tempo de duração da animação.

    easingIndica qual função do plugin easing usar na transição.

    callbackFunção a ser chamada ao finalizar a animação.

Sem nenhum parâmetro, o método .hide() vai simplesmente ocultar o elemento:

$('.alvo').hide();

Os elementos alvo serão ocultos automaticamente, sem nenhum animação. Isso é aproximadamente o mesmo que utilizar .css('display', 'none'), exceto que o valor da propriedade display é salva no jQuery's Data Cache assim o display pode ser restaurado posteriormente ao seu valor inicial. Se um elemento tem display com valor inline, então ele será oculto e mostrado novamente de forma inline.

Quando uma duração é informada, .hide() se torna uma função de animação. O método .hide() anima a largura, altura e opacidade do elemento alvo simultâneamente. Quando essas propriedades chegam a 0, a propriedade display é determinada como none para assegurar que o elemento não afetará mais o layout da página.

A duração é informada em milisegundos; Valores altos indicam animações mais lentas e não mais rápidas. Textos como 'fast' e 'slow' podem ser informados para indicar durações de 200 e 600 milisegundos, respectivamente.

A partir da jQuery 1.4.3, uma opção chamada de função easing passou a poder ser usada. Funções easing determinam a velocidade que a animação irá progredir em diferentes momentos dentro da mesma. A única implementação da easing na jQuery são somente as padrões, chamada swing, e outra que vem progredindo em constante ritmo de paz, chamada linear. Mais funções easing estão disponíveis com o uso de plugins, mais notavelmente a jQuery UI suite.

Se for informado o callback é chamado assim que a animação é finalizada. Isto pode ser útil para unir animações diferentes em uma sequência. O callback não envia nenhum argumento, porém this é definido como o elemento DOM sendo animado. Se vários elementos estão sendo animados, é importante notar que o callback é executado uma vez por elemento correspondente e não para a animação como um todo.

Podemos animar qualquer elemento, por exemplo uma simples imagem:

<div id="clickme">
  Clique Aqui
</div>
<img id="livro" src="book.png" alt="" width="100" height="123" />
Com o elemento sendo apresentado, nós podemos lentamente ocultá-lo:
$('#clickme').click(function() {
  $('#livro').hide('slow', function() {
    alert('Animação Completa.');
  });
});

Nota Adicional:

  • Todos os efeitos da jQuery, incluíndo .hide(), podem ser globalmente desativados pela configuração jQuery.fx.off = true, que efetivamente determina a duração como 0. Para mais informações, veja jQuery.fx.off.

Exemplos:

Exemplo: Oculta todos os parágrafos e o link ao ser clicado. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Olá</p>
  <a href="#">Clique para me ocultar também</a>
  <p>Aqui vem outro parágrafo</p>
<script>

    $("p").hide();
    $("a").click(function () {
      $(this).hide();
      return true;
    });
</script>

</body>
</html>

Exemplo: Anima todos os spans (palavras nesse caso) para ocultar-se rapidamente, completando cada animação em 200 milisegundos. Assim que uma animação termina, se inicia a próxima. 

<!DOCTYPE html>
<html>
<head>
  <style>
  span { background:#def3ca; padding:3px; float:left; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="hidr">Ocultar</button>
  <button id="showr">Mostrar</button>
  <div>

    <span>Once</span> <span>upon</span> <span>a</span> 
    <span>time</span> <span>there</span> <span>were</span> 
    <span>three</span> <span>programmers...</span>

  </div>
<script>
    $("#hidr").click(function () {
      $("span:last-child").hide("fast", function () {
        // use callee portanto não terá que nomear a função
        $(this).prev().hide("fast", arguments.callee); 
      });
    });
    $("#showr").click(function () {
      $("span").show(2000);
    });

</script>

</body>
</html>

Exemplo: Oculta as divs quando clicadas em mais de 2 seconds, então remove o elemento div quando estiver oculto. Tente clicar em mais de um box por vez. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background:#ece023; width:30px; 
        height:40px; margin:2px; float:left; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div></div>
<script>
    for (var i = 0; i < 5; i++) {
      $("<div>").appendTo(document.body);
    }
    $("div").click(function () {
      $(this).hide(2000, function () {
        $(this).remove();
      });
    });
</script>

</body>
</html>
