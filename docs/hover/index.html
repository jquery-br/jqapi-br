.hover( handlerIn(eventObject), handlerOut(eventObject) ) Retorna: jQuery

Descrição: Atribui dois manipuladores para os elementos encontrados, para serem executados quando o ponteiro do mouse aponta ou sai de cima do elemento.

  • adicionado na versão: 1.0.hover( handlerIn(eventObject), handlerOut(eventObject) )

    handlerIn(eventObject)Uma função para executar quando o ponteiro do mouse aponta o elemento.

    handlerOut(eventObject)Uma função para executar quando o ponteiro do mouse sai de cima do elemento.

O método .hover() atribui manipuladores para ambos os eventos mouseenter e mouseleave. Você pode usá-lo para simplesmente aplicar comportamento a um elemento durante o tempo que o mouse está sobre o elemento.

Chamando $(selector).hover(handlerIn, handlerOut) é um atalho para:

$(selector).mouseenter(handlerIn).mouseleave(handlerOut);

Veja as discussões para .mouseenter() e .mouseleave() para mais detalhes.

Exemplos:

Exemplo: Para adicionar um estilo especial para a lista de itens que estão sendo pairados sobre eles, tente: 

<!DOCTYPE html>
<html>
<head>
  <style>
  ul { margin-left:20px; color:blue; }
  li { cursor:default; }
  span { color:red; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <ul>
    <li>Milk</li>
    <li>Bread</li>
    <li class='fade'>Chips</li>

    <li class='fade'>Socks</li>
  </ul>
<script>
$("li").hover(
  function () {
    $(this).append($("<span> ***</span>"));
  }, 
  function () {
    $(this).find("span:last").remove();
  }
);



//li with fade class
$("li.fade").hover(function(){$(this).fadeOut(100);$(this).fadeIn(500);});

</script>

</body>
</html>

Exemplo: Para adicionar um estilo especial para células de tabela que estao sendo pairados sobre, tente: 

$("td").hover(
  function () {
    $(this).addClass("hover");
  },
  function () {
    $(this).removeClass("hover");
  }
);

Exemplo: Para desatribuir o exemplo acima, use: 

$("td").unbind('mouseenter mouseleave');

.hover( handlerInOut(eventObject) ) Retorna: jQuery

Descrição: Atribui um único manipulador para os elementos encotrados, para executarem quando o ponteiro do mouse aponta ou sai de cima dos elementos.

  • adicionado na versão: 1.4.hover( handlerInOut(eventObject) )

    handlerInOut(eventObject)Uma função para executar quando o ponteiro do mouse aponta ou sai do elemento.

O método .hover(), quando passado para uma única função, executará esse manipulador para ambos os eventos mouseenter e mouseleave. Isso permite ao usuário usar os vários métodos de alternância dentro do manipulador ou para responder diferentemente nele dependendo do event.type.

Chamando $(selector).hover(handlerInOut) é um atalho para:

$(selector).bind("mouseenter mouseleave", handlerInOut);

Veja as discussões para .mouseenter() e .mouseleave() para mais detalhes.

Exemplo:

Deslize o próximo irmão LI para cima ou para baixo no hover, e alterne uma classe.

<!DOCTYPE html>
<html>
<head>
  <style>
  ul { margin-left:20px; color:blue; }
  li { cursor:default; }
  li.active { background:black;color:white; }
  span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <ul>
    <li>Leite</li>
    <li>Branco</li>
    <li>Cenouras</li>
    <li>Laranja</li>
    <li>Brócoli</li>
    <li>Verde</li>
  </ul>
<script>
$("li")
.filter(":odd")
.hide()
 .end()
.filter(":even")
.hover(
  function () {
    $(this).toggleClass("active")
      .next().stop(true, true).slideToggle();
  }
);


</script>

</body>
</html>
