Seletor ID (“#id”)

Categorias: Seletores > Básico

Seletor id

adicionado na versão: 1.0jQuery('#id')

  • id
    Um ID a ser buscado, especificado através do atributo ID de um elemento.

Descrição: Seleciona apenas um elemento com o atributo ID dado.

Para o seletor id, o jQuery usa a função do JavaScript document.getElementById(), o que é extremamente eficiente. Quando um outro seletor é adicionado junto ao seletor id, como h2#pageTitle, o jQuery executa uma verificação adicional antes de identificar o element como resultado.

Como sempre, relembre que como desenvolvedor, seu tempo é normalmente o recurso mais valioso. Não se foque na otimização da velocidade dos seletores a não ser que esteja claro que a performance precisa ser melhorada.

CADA ID DEVE SER USADO APENAS UMA VEZ DENTRO DO DOCUMENTO. Caso exista mais de um elemento com o mesmo id, pesquisas que use esse ID irão retornar apenas o primeiro elemento no DOM. Esse comportamento não pode ser confiável, entretanto; um documento com mais de um elemento com o mesmo ID é inválido.

Se o id contem caracteres como If the id contains characters como dois pontos você deve escapar esses caracteres com barra invertida.

Exemplo: Localiza o elemento com o id "myDiv". 

<!DOCTYPE html>
<html>
<head>
  <style>
  div {
    width: 90px;
    height: 90px;
    float:left;
    padding: 5px;
    margin: 5px;
    background-color: #EEEEEE;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div id="notMe"><p>id="notMe"</p></div>

  <div id="myDiv">id="myDiv"</div>
<script>$("#myDiv").css("border","3px solid red");</script>

</body>
</html>

Exemplo: Localiza o elemento com o id "myID.entry[1]". Veja como certos caracteres deve ser escapados com barra invertida. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div {
    width: 300px;
    float:left;
    padding: 2px;
    margin: 3px;
    background-color: #EEEEEE;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div id="myID.entry[0]">id="myID.entry[0]"</div>

  <div id="myID.entry[1]">id="myID.entry[1]"</div>
  <div id="myID.entry[2]">id="myID.entry[2]"</div>
<script>$("#myID\\.entry\\[1\\]").css("border","3px solid red");</script>

</body>
</html>
