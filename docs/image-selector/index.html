Seletor :image

Categorias: Seletores > Formulários

Seletor image

adicionado na versão: 1.0jQuery(':image')

Descrição: Seleciona todos os elementos do tipo image.

Exemplo:

Localiza todos os inputs do tipo image.

<!DOCTYPE html>
<html>
<head>
  <style>
  textarea { height:45px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>
    <input type="button" value="Botão"/>
    <input type="checkbox" />

    <input type="file" />
    <input type="hidden" />
    <input type="image" />

    <input type="password" />
    <input type="radio" />
    <input type="reset" />

    <input type="submit" />
    <input type="text" />
    <select><option>Option<option/></select>

    <textarea></textarea>
    <button>Botão</button>
  </form>
  <div>
  </div>
<script>
    var input = $("input:image").css({background:"yellow", border:"3px red solid"});
    $("div").text("Para esse tipo o jQuery encontrou " + input.length + ".")
            .css("color", "red");
    $("form").submit(function () { return false; }); // evita o submit

</script>

</body>
</html>
