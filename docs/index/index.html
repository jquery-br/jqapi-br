.index()

Categories: Miscellaneous > DOM Element Methods

.index( ) Retorna: Number

Descrição: Procura por um dado elemento dentre os elementos encontrados.

  • adicionado na versão: 1.4.index()

  • adicionado na versão: 1.4.index( seletor )

    selectorUm seletor que representa uma coleção do jQuery na qual procurar por um elemento.

  • adicionado na versão: 1.0.index( elemento )

    elementoUm elemento DOM ou o primeiro elemento dentro do objeto jQuery para procurar.

Valores Retornados

Se nenhum argumento for passado para o método .index(), o valor de retorno vai ser um inteiro indicando a posição do primeiro elemento dentro dentro do objeto jQuery relativo aos seus elementos irmãos.

Se o método .index() for chamado numa coleção de elementos e um elemento DOM ou um objeto jQuery for passado, .index() retornará um inteiro indicando a posição do elemento passado relativo a coleção original.

Se um seletor for passado como argumento, .index() retornará um inteiro indicando a posição do elemento original relativa aos elementos encontrados pelo seletor. Se nenhum elemento for encontrado, .index() retornará -1.

Detalhe

A operação complementar de .get(), que aceita e retorna um nó DOM, .index() pode aceitar um nó DOM e retornar um índice. Suponha que tenhamos uma lista não ordenada simples na página:

<ul>
  <li id="foo">foo</li>
  <li id="bar">bar</li>
  <li id="baz">baz</li>
</ul>

Se recuperarmos um dos três itens da lista (Por exemplo, através de uma função DOM ou como contexto de um handler de evento), .index() pode procurar por este item de lista dentro do conjunto de elementos encontrados:

var listItem = document.getElementById('bar');
alert('Indice: ' + $('li').index(listItem));

Nós recebemos de volta a posição( baseada em 0 ) do item da lista:

Indice: 1

Similarmente, se nós recuperarmos um objeto jQuery que consiste em um dos três itens da lista, .index() vai procurar por este item da lista:

var listItem = $('#bar');
alert('Indice: ' + $('li').index(listItem));

Nós recebemos de volta a posição( baseada em 0 ) do item da lista:

Indice: 1

Note que se a coleção jQuery usada como argumento para o método .index() contiver mais de um elemento, o primeiro elemento dentro do conjunto encontrado de elementos vai ser usado.

var listItems = $('li:gt(0)');
alert('Indice: ' + $('li').index(listItems));

Nós recebemos de volta a posição( baseada em 0 ) do primeiro item da lista dentro do conjunto encontrado:

Indice: 1

Se usarmos uma string como o argumento do método .index() , ela vai ser interpretada como uma string seletora do jQuery. O primeiro elemento dentre os objetos encontrados que também obedece ao seletor é então localizado.

var listItem = $('#bar');
alert('Indice: ' + listItem.index('li'));

Nós recebemos de volta a posição( baseada em 0 ) do item da lista:

Indice: 1

Se omitirmos o argumento, .index() vai retornar a posição do primeiro elemento dentro do conjunto de elementos encontrados em relação aos seus irmãos:

alert('Indice: ' + $('#bar').index();

Novamente, nós recebemos de volta a posição( baseada em 0 ) do item da lista::

Indice: 1

Exemplos:

Exemplo: Ao clicar, retorna o índice(baseado em 0) da div clicada na página. 

<!DOCTYPE html>
<html>
<head>
  <style>
div { background:yellow; margin:5px; }
span { color:red; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <span>Clique em uma div!</span>
<div>Primeira div</div>
<div>Segunda div</div>
<div>Terceira div</div>
<script>
$("div").click(function () {
  // Este é o elemento dom clicado
  var indice = $("div").index(this);
  $("span").text("Esta foi a div de numero:" + indice);
});
</script>

</body>
</html>
