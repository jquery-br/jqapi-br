jQuery.ajaxSetup()

Categorias: Ajax > Low-Level Interface

jQuery.ajaxSetup( options )

Descrição: Atribui valores padrão para futuras requisições Ajax.

  • adicionado na versão: 1.1 jQuery.ajaxSetup( options )

    options Um conjunto de pares key/value que configura uma requisição padrão Ajax. Todas opções são opcionais.

Para detalhes das configurações disponíveis para $.ajaxSetup(), veja $.ajax().

Todas calls Ajax subsequentes usando qualquer função irão usar as novas configurações, a não ser que sejam substituídas por calls individuais, até a próxima invocação do $.ajaxSetup().

Por exemplo, a seguir atribuímos um um valor padrão para o parâmetro url antes de fazer ping repetidamente no servidor:

$.ajaxSetup({
  url: 'ping.php'
});

Agora, cada vez que uma requisição Ajax é realizada, a URL "ping.php" será usada automaticamente:

$.ajax({
  // url not set here; uses ping.php
  data: {'name': 'Dan'}
});

Nota: Funções globais de callback devem ser configuradas com seus respectivos métodos do event handler global Ajax —.ajaxStart(), .ajaxStop(), .ajaxComplete(), .ajaxError(), .ajaxSuccess(), .ajaxSend()— e não dentro do objeto options para $.ajaxSetup().

Exemplo:

Atribui valores padrão para requisições Ajax à url "/xmlhttp/", desabilita handlers globais e usa POST ao invés de GET. A requisição Ajax a seguir envia alguns dados sem ter que configurar mais nada.

$.ajaxSetup({
   url: "/xmlhttp/",
   global: false,
   type: "POST"

 });
 $.ajax({ data: myData });
