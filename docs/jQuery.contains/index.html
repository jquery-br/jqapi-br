jQuery.contains()

Categorias: Utilities

jQuery.contains( container, contained ) Retorna: Boolean

Descrição: Verifica se um nó do DOM esta dentro de outro nó DOM.

  • adicionado na versão: 1.4jQuery.contains( container, contained )

    containerO elemento DOM que pode ter o outro elemento.

    containedO elemento DOM que pode estar dentro do outro elemento

Exemplo:

Verifica se um elemento esta dentro de outro

.

jQuery.contains(document.documentElement, document.body); // true
jQuery.contains(document.body, document.documentElement); // false
