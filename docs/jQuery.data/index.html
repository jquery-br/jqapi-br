jQuery.data( element, key, value ) Retorna: Object

Descrição: Armazena dados arbitrários associados a um grupo de elementos. Retorna o valor que estava setado.

  • adicionado na versão: 1.2.3jQuery.data( element, key, value )

    elementO elemento DOM que irá se associar com os dados.

    keyUma string nomeando um pedaço de dados a ser armazenado.

    valueO novo valor.

Note:Esse é um méotodo low-level; um mais conveniente .data() tabém está disponível.

O método .data() permite anexar dados a qualquer tipo de elementos DOM de uma maneira segura a referências circulares e consequentemente a leaks de memória. O jQuerys assegura que os dados são removidos quando os elementos DOM são removidos via métodos jQuery, e quando o usuário sai da página. Podemos inserir diversos valores distintos para um único elemento e retorná-los depois:

jQuery.data(document.body, 'foo', 52);
jQuery.data(document.body, 'bar', 'test');

Note que esse método atualmente não provê suporte cross-platform para setar dados em documentos XML, como o Internet Explorer nao premite adicionar dados via propriedades expando.

Exemplo:

Armazenar e retornar um valor de um elemento div.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { color:blue; }
  span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>
    O valor armazenado era
    <span></span>
    e
    <span></span>
  </div>
<script>var div = $("div")[0];
    jQuery.data(div, "test", { first: 16, last: "pizza!" });
    $("span:first").text(jQuery.data(div, "test").first);
    $("span:last").text(jQuery.data(div, "test").last);</script>

</body>
</html>

jQuery.data( element, key ) Retorna: Object

Descrição: Retorna o valor armazenado para o elemento que foi setado por jQuery.data(element, name, value), ou todos os dados armazenados para o elemento.

  • adicionado na versão: 1.2.3jQuery.data( element, key )

    elementO elemento DOM que ira buscar os dados.

    keyNome do dado armazenado.

  • adicionado na versão: 1.4jQuery.data( element )

    elementO elemento DOM que ira buscar os dados.

Note:Esse é um méotodo low-level; um mais conveniente .data() tabém está disponível.

Preposição data-* do atributo HTML5: Esse método low-level não retorna o atributo data-* caso o método .data() já não o tenha retornado.

O método jQuery.data() permite anexar dados a qualquer tipo de elementos DOM de uma maneira segura a referências circulares e consequentemente a leaks de memória. Podemos recuperar diversos valores distintos para um elemento de cada vez, assim como setar.

alert(jQuery.data( document.body, 'foo' ));
alert(jQuery.data( document.body ));

As linhas acima da um alert com o valor que foi setado no elemento body. Se nada foi setado, uma string vazia é retornada.

Chamando jQuery.data(element) retorna todos os valores como um objeto JavaScript. Note que o jQuery usa esse método para gurdar dados para uso interno, como manipuladores de eventos, então, não assuma que isso contém apenas dados que seu código armazenou.

Note que esse método atualmente não provê suporte cross-platform para setar dados em documentos XML, como o Internet Explorer nao premite adicionar dados via propriedades expando.

Exemplo:

Pega o dado nomeado "blah" armazenado em um elemento.

<!DOCTYPE html>
<html>
<head>
  <style>
div { margin:5px; background:yellow; }
button { margin:5px; font-size:14px; }
p { margin:5px; color:blue; }
span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>A div</div>
<button>Pega "blah" da div</button>
<button>Coloca "Olá" em "blah"</button>

<button>Coloca 86 em "blah"</button>
<button>Remove "blah"  da div</button>
<p>O valor "blah" na div é <span>?</span></p>
<script>
$("button").click(function(e) {
  var value, div = $("div")[0];

  switch ($("button").index(this)) {
    case 0 :
      value = jQuery.data(div, "blah");
      break;
    case 1 :
      jQuery.data(div, "blah", "hello");
      value = "Stored!";
      break;
    case 2 :
      jQuery.data(div, "blah", 86);
      value = "Stored!";
      break;
    case 3 :
      jQuery.removeData(div, "blah");
      value = "Removed!";
      break;
  }

  $("span").text("" + value);
});

</script>

</body>
</html>
