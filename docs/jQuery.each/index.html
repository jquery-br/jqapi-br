jQuery.each()

Categories: Utilities

jQuery.each( collection, callback(indexInArray, valueOfElement) ) Returns: Object

Description: Uma função genérica de interação, que pode ser usada para interagir sobre objetos e arrays. Arrays e arrays simulados com um apropriedade length (tal como argumentos de uma função) são inteirados com um indice numérico, de 0 à lengh-1. Outros objetos são inteirados pelos nomes de suas propriedades.

  • versão: 1.0 jQuery.each( collection, callback(indexInArray, valueOfElement) )

    collection O objeto ou array da interação.

    callback(indexInArray,
    valueOfElement)     Função que será executada para todo objeto.

A função $.each() não é igual a .each() , que é usada para interagir, exclusivamente, em objetos jQuery. A função $.each() pode ser usada para interagir sobre qualquer coleção, seja um map (objecto JavaScript) ou um array. No caso de um array, o callback passa o indice do array e um correspondente ao valor do array em cada interação. (O valor também pode ser acessado através da palavra chave this, but Javascript sempre envolverá o valor de this como um Object mesmo se o mesmo for uma simples string or um valor numérico.) O método retorna como primeiro argumento, o objeoto que foi inteirado. 

$.each([52, 97], function(index, value) { alert(index + ': ' + value); });

Isso produz duas mensagens no alert:

0: 52
1: 97

Se um map é usado como uma collection, o callback passa um par chave-valor em cada interação: 

var map = { 'flammable': 'inflammable', 'duh': 'no duh' };

$.each(map, function(key, value) { alert(key + ': ' + value); });

Novamente, isso produz duas mensagens:

flammable: inflammable
duh: no duh

Podemos parar o loop $.each() numa interação em particular fazendo com que o a função callback retorne false. Retornando non-false é o mesmo que afirmarmos continue em um loop for; Isso irá imediatamente para a próxima interação.

Exemplos:

Exemplo: Percorre a matriz exibindo cada número tanto como uma palavra ou numeral 

<!DOCTYPE html>
<html>
  <head>
    <style> 
      div { 
        color:blue;
      } 
      div#five {
        color:red;
      } 
    </style>

    <script src="http://code.jquery.com/jquery-git.js"></script>
  </head>

  <body> 
    <div id="one"></div> 
    <div id="two"></div>
    <div id="three"></div>
    <div id="four"></div>
    <div id="five"></div> 

    <script> 
      var arr = [ "one", "two", "three", "four", "five" ];
      var obj = { one:1, two:2, three:3, four:4, five:5 };
      
      jQuery.each(arr, function() { 
        $("#" + this).text("Mine is " + this + ".");
        
        return (this != "three"); // will stop running after "three" 
      });

      jQuery.each(obj, function(i, val) { 
          $("#" + i).append(document.createTextNode(" - " + val));
      });
    </script>
  </body>
</html>

Exemplo: Interage sobre os itens em um array, acessando o item atual e seu índice. 

$.each( ['a','b','c'], 
  function(i, l){ alert( "Index #" + i + ": " + l );
});

Exemplo: Interage sobre as propriedades de um objeto, acessando tanto o ítem atual quanto sua chave. 

$.each( { name: "John", lang: "JS" },
  function(k, v){ alert( "Key: " + k + ", Value: " + v );
});
