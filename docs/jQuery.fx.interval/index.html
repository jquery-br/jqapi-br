jQuery.fx.interval

Categorias: Effects > Custom | Properties > Properties of the Global jQuery Object

jQuery.fx.interval Retorna: Number

Descrição: A taxa (em milisegundos) no disparo da animação.

  • adicionado na versão: 1.4.3jQuery.fx.interval

Essa propriedade pode ser manipulada para ajustar o número de frames por segundo que a animação irá rodar. O padrão é 13 milisegundos. Deixando esse número baixo poderia fazer as animações rodar mais suave em navegadores rápidos (como o Chrome) mas pode ter implicações com a performance e CPU fazendo isso.

Desde que o jQuery usa apenas um intervalo global, nenhuma animação deve estar rodando ou todas as aimações devem parar para a mudança dessa propriedade fazer efeito.

Exemplo:

Força todas as animações rodarem com menos frames.

<!DOCTYPE html>
<html>
<head>
  <style>
    div { width:50px; height:30px; margin:5px; float:left;
          background:green; }
    </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p><input type="button" value="Run"/></p>
<div></div>
<script>
jQuery.fx.interval = 100;

$("input").click(function(){
  $("div").toggle( 3000 );
});
  </script>

</body>
</html>
