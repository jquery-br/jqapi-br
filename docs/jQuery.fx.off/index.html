jQuery.fx.off

Categorias: Effects > Custom | Properties > Properties of the Global jQuery Object

jQuery.fx.off Retorna: Boolean

Descrição: Desabilita todas as animações globalmente.

  • adicionado na versão: 1.3jQuery.fx.off

Quando essa propriedade é setada para true, todos os métodos de animação quando chamados vão setar imediatamente os elementos ao seu estado final, ao invés de mostrar o efeito. Isso pode ser desejado por algumas rasões:

  • jQuery é usado em um deviser com poucos recursos.
  • Usuários estão encontrando problemas de acessibilidade com as animações (veja o artigo Desligando animações para mais informações).

As animações podem voltar setando a propriedade para false.

Exemplo:

Ligar e desligar animações

<!DOCTYPE html>
<html>
<head>
  <style>
    div { width:50px; height:30px; margin:5px; float:left;
          background:green; }
    </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p><input type="button" value="Run"/> <button>Toggle fx</button></p>
<div></div>
<script>
var toggleFx = function() {
  $.fx.off = !$.fx.off;
};
toggleFx();

$("button").click(toggleFx)

$("input").click(function(){
  $("div").toggle("slow");
});
  </script>

</body>
</html>
