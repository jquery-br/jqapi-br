jQuery.getScript()

Categorias: Ajax > Shorthand Methods

jQuery.getScript( url, [ success(data, textStatus) ] ) Retorna: XMLHttpRequest

Descrição: Carrega um arquivo JavaScript do servidor usando uma requisição GET HTTP, então o executa.

  • Adicionado na versão: 1.0jQuery.getScript( url, [ success(data, textStatus) ] )

    url Uma string contendo a URL à qual a requisição será enviada .

    success(data, textStatus) Uma função callback que é executada se a requisição for bem sucedida.

>Essa é uma simplificação de uma função Ajax, que é equivalente a:

$.ajax({
  url: url,
  dataType: 'script',
  success: success
});

O arquivo JavaScript é passado ao callback. Isso geralmente não é útil pois o script já foi executado neste ponto.

O script é executa em um contexto global, então ele pode se referir a outras variáveis e usar funções JQuery. Scripts incluídos devem ter algum impacto sobre a página atual:

$('.result').html('<p>Lorem ipsum dolor sit amet.</p>');

O script pode então ser incluído e executado referenciando o nome do arquivo:

$.getScript('ajax/test.js', function() {
  alert('Load was performed.');
});

Exemplos:

Exemplo: Carrega o plugin JQuery Color Animation oficial e faz com que algumas animações de cor ocorram assim que a funcialidade é carregada. 

<!DOCTYPE html>
<html>
<head>
  <style>.block { 
   background-color: blue; 
   width: 150px; 
   height: 70px;
   margin: 10px; 
}</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="go">&raquo; Run</button>

<div class="block"></div>

<script>$.getScript("http://dev.jquery.com/view/trunk/plugins/color/jquery.color.js", function(){
  $("#go").click(function(){
    $(".block").animate( { backgroundColor: 'pink' }, 1000)
      .animate( { backgroundColor: 'blue' }, 1000);
  });
});</script>

</body>
</html>
