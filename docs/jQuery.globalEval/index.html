jQuery.globalEval()

Categorias: Utilities

jQuery.globalEval( code )

Descrição: Executa algum código JavaScript globalmente.

  • adicionado na versão: 1.0.4jQuery.globalEval( code )

    codeO código JavaScript a ser executado.

Esse método se comporta diferentemente do normal eval() do JavaScript, em que é executado dentro do contexto global (o que é importante para carregar scripts externos dinamicamente).

Exemplo:

Executa um script no contexto global.

function test(){
    jQuery.globalEval("var newVar = true;")
}
test();
// newVar === true
