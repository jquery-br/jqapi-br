jQuery.hasData()

Categorias: Data

jQuery.hasData( element ) Retorna: Boolean

Descrição: Determina se um elemento tem algum dado jQuery associado a ele.

  • adicionado na versão: 1.5jQuery.hasData( element )

    elementO elemento DOM a ser checado.

O método jQuery.hasData() prove um meio de determinar se um elemento tem valores setador através do jQuery.data(). Se não foi encontrado nenhum dado associado ao elemento (não há dados ou esta vaziu), o método retorna false; caso contrário retorna true.

A primeira vantagem do jQuery.hasData(element) é que ele não cria e associa um dado ao objeto caso atualmente não exista. Em contraste, jQuery.data(element) sempre retorna um objeto de dados para a chamada, criando um caso previamente não exista o objeto.

Exemplo:

Colocar dados em um elemento e ver o resultado do hasData.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Results: </p>
<script>
$(function(){
  var $p = jQuery("p"), p = $p[0];
  $p.append(jQuery.hasData(p)+" "); /* false */
  jQuery.data(p, "testando", 123);
  $p.append(jQuery.hasData(p)+" "); /* true*/
  jQuery.removeData(p, "testando");
  $p.append(jQuery.hasData(p)+" "); /* false */
});
</script>

</body>
</html>
