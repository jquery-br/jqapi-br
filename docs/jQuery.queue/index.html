jQuery.queue( element, [ queueName ] ) Retorna: Array

Descrição: Mostra a fila de funções a serem executadas para os elementos selecionados.

  • adicionado na versão: 1.3jQuery.queue( element, [ queueName ] )

    elementUm elemento do DOM para inspecionar uma fila.

    queueNameUma string contendo o nome da fila. Padrão para fx, a fila padrão para efeitos.

Nota: Este é um método low-level, você deveria provavelmente usar .queue() ao invés.

Exemplo:

Mostra o tamanho da fila.

<!DOCTYPE html>
<html>
<head>
  <style>div { margin:3px; width:40px; height:40px;
        position:absolute; left:0px; top:30px;
        background:green; display:none; }
  div.newcolor { background:blue; }
  span { color:red; }  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="show">Mostra o tamanho da fila</button>
  <span></span>
  <div></div>
<script>$("#show").click(function () {
      var n = jQuery.queue( $("div")[0], "fx" );
      $("span").text("O tamanho é: " + n.length);
    });
    function runIt() {
      $("div").show("slow");
      $("div").animate({left:'+=200'},2000);
      $("div").slideToggle(1000);
      $("div").slideToggle("fast");
      $("div").animate({left:'-=200'},1500);
      $("div").hide("slow");
      $("div").show(1200);
      $("div").slideUp("normal", runIt);
    }
    runIt();</script>

</body>
</html>

jQuery.queue( element, queueName, newQueue ) Retorna: jQuery

Descrição: Manipula a fila de funções a ser executada para os elementos selecionados.

  • adicionado na versão: 1.3jQuery.queue( element, queueName, newQueue )

    elementUm elemento DOM onde o array de funções de fila é anexado.

    queueNameUma string contendo o nome da fila. Padrão para fx, a fila padrão para efeitos.

    newQueueUm array de funções para substituir o atual conteúdo da fila.

  • adicionado na versão: 1.3jQuery.queue( element, queueName, callback() )

    elementA DOM element on which to add a queued function.

    queueNameUma string contendo o nome da fila. Padrão para fx, a fila padrão para efeitos.

    callback()A nova função a ser adicionada na fila.

Nota: Este é um método low-level, você deveria provavelmente usar .queue() ao invés.

Todo elemento pode ter um ou mais filas de funções anexadas pelo jQuery. Na maioria das aplicações, apenas uma fila (chamada fx) é usada. Filas permitem uma sequencia de ações a serem chamada em um elemento assincronamente, sem interromper a execução do programa.

O método jQuery.queue() permite manipular diretamente essa fila de funções. Chamando jQuery.queue() om um callback é particularmente útil; Isso nos permite colocar uma nova função no fim da fila.

Note que adicionando uma função com jQuery.queue(), devemos garantir que o jQuery.dequeue() é eventualmente chamado, assim a próxima função na linha é executada.

Examples:

Exemplos: Enfileira uma função personalizada. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:3px; width:40px; height:40px;
        position:absolute; left:0px; top:30px;
        background:green; display:none; }
  div.newcolor { background:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Clique aqui...
  <div></div>
<script>
   $(document.body).click(function () {
      $("div").show("slow");
      $("div").animate({left:'+=200'},2000);
      jQuery.queue( $("div")[0], "fx", function () {
        $(this).addClass("newcolor");
        jQuery.dequeue( this );
      });
      $("div").animate({left:'-=200'},500);
      jQuery.queue( $("div")[0], "fx", function () {
        $(this).removeClass("newcolor");
        jQuery.dequeue( this );
      });
      $("div").slideUp();
    });</script>

</body>
</html>

Exemplo: Seta um array para deletar a fila. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:3px; width:40px; height:40px;
        position:absolute; left:0px; top:30px;
        background:green; display:none; }
  div.newcolor { background:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>

  <button id="start">Iniciar</button>
  <button id="stop">Parar</button>
  <div></div>
<script>
   $("#start").click(function () {
      $("div").show("slow");
      $("div").animate({left:'+=200'},5000);
      jQuery.queue( $("div")[0], "fx", function () {
        $(this).addClass("newcolor");
        jQuery.dequeue( this );
      });
      $("div").animate({left:'-=200'},1500);
      jQuery.queue( $("div")[0], "fx", function () {
        $(this).removeClass("newcolor");
        jQuery.dequeue( this );
      });
      $("div").slideUp();
    });
    $("#stop").click(function () {
      jQuery.queue( $("div")[0], "fx", [] );
      $("div").stop();
    });
</script>

</body>
</html>
