jQuery.removeData()

Categorias: Data | Utilities

jQuery.removeData( element, [ name ] ) Retorna: jQuery

Descrição: Remove um pedaço de dados previamente guardado.

  • adicionado na versão: 1.2.3jQuery.removeData( element, [ name ] )

    elementUm elemento DOM do qual o dado vai ser removido.

    nameUma string nomeando o pedaço dos dados para deletar.

Nota: Esse é um método de baixo nível, você provavelmente deveria usar .removeData() ao invés.

O método .removeData() nos permite remover valores que foram previamente setados usando .data(). Quando chamamos com o nome da chave, o .removeData() deleta esse valor em particular; quando chamamos sem argumentos, todos os valores são removidos.

Exemplo:

Definir dados armazenados para 2 nomes e após remover um deles.

<!DOCTYPE html>
<html>
<head>
  <style>
		div { margin:2px; color:blue; }
		span { color:red; }
		</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>value1 before creation: <span></span></div>
		<div>value1 depois da criação: <span></span></div>
		<div>value1 depois de remover: <span></span></div>
		<div>value2 depois de remover: <span></span></div>
<script>
var div = $("div")[0];
$("span:eq(0)").text("" + $("div").data("test1"));
jQuery.data(div, "test1", "VALUE-1");
jQuery.data(div, "test2", "VALUE-2");
$("span:eq(1)").text("" + jQuery.data(div, "test1"));
jQuery.removeData(div, "test1");
$("span:eq(2)").text("" + jQuery.data(div, "test1"));
$("span:eq(3)").text("" + jQuery.data(div, "test2"));</script>

</body>
</html>
