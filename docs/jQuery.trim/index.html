jQuery.trim()

Categorias: Utilitários

jQuery.trim( str ) Retorna: String

Descrição: Remover o espaço em branco do início e no final de uma string.

  • versão adicionada: 1.0jQuery.trim( str )

    strA string para usar o trim.

A função $.trim() remove todas as quebras de linha, espaços (incluindo espaços sem quebras de linha), e abas desde o início ao final de uma string fornecida. Se esses caracteres em branco estiverem no meio da string, eles permanecer√£o.

Exemplos:

Exemplo: Remover os dois espaços em branco no início e no final da string. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Exibir o exemplo do Trim</button>
<script>

$("button").click(function () {
var str = "     vários espaços antes e depois     ";
alert("'" + str + "'");

str = jQuery.trim(str);
alert("'" + str + "' - não há mais espaços");
});

</script>

</body>
</html>

Demonstração:

Exemplo: Remover os dois espaços em branco no início e no final da string. 

$.trim("  Oi, como vai você?  ");
