jQuery.unique()

Categories: Utilities

jQuery.unique( array ) Retorna: Array

Descrição: Ordena um array de elementos DOM, e remove os duplicados. Note que isso funciona apenas para elementos do DOM, não funciona com strings ou números.

  • adicionado na versão: 1.1.3jQuery.unique( array )

    arrayO Array de elementos do DOM.

A função $.unique() procura através de um array de objetos, ordenando o array e removendo qualquer nó duplicado. Essa função só funciona com array plenos de JavaScript de elementos do DOM e é principalmente usado internamente pelo jQuery.

O resultado sempre será retornado na ordem do documento.

Exemplo:

Remove qualquer elemento duplicado do array de divs.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { color:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>There are 6 divs in this document.</div>
  <div></div>
  <div class="dup"></div>
  <div class="dup"></div>

  <div class="dup"></div>
  <div></div>
<script>

    var divs = $("div").get(); // unique() precisa pegar um array nativo

    // adicionad 3 elementos com a classe dup (eles são divs)
    divs = divs.concat($(".dup").get());
    $("div:eq(1)").text("Antes do unique são: " + divs.length + " elementos.");

    divs = jQuery.unique(divs);
    $("div:eq(2)").text("Depois do unique são: " + divs.length + " elementos.")
                  .css("color", "red");

</script>

</body>
</html>
