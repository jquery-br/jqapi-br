.keypress()

Categories: Events > Keyboard Events

.keypress( handler(eventObject) ) Retorna: jQuery

Descrição: Acopla um manipulador de evento para o evento JavaScript "keypress", ou dispara esse evento a um elemento.

  • adicionado na versão: 1.0.keypress( handler(eventObject) )

    handler(eventObject)Uma função para executar a cada vez que o evento é disparado.

  • adicionado na versãao: 1.4.3.keypress( [ eventData ], handler(eventObject) )

    eventDataUm mapa de dados que será passado ao manipulador de evento.

    handler(eventObject)Uma funçãao a ser executada a cada vez que o evento é disparado.

  • adicionado na versão: 1.0.keypress()

Note: como o evento keypress não cobre especificação oficial, o comportamento real encontrado quando o usa pode ser diferente conforme os browsers, versão de browsers e plataformas.

Este método é um atalho para .bind('keypress', handler) nas duas primeiras variações, e .trigger('keypress') no terceiro.

O evento keypress é enviado a um elemento quando o browser registra a entrada via teclado. Isto é similar ao evento keydown , exceto no caso de emq ue a tecla se repete. Se o usuário pressiona e segura uma tecla, um evento keydown é disparado uma vez, mas eventos keypress separados são disparados para cada caractere inserido. Em adição, os eventos modificadores de teclas (como Shift) disparam keydown eventos exceto keypress.

Um manipulador de evento keypress pode ser acoplado a qualquer elemento, mas o evento é enviado apenas para o elemento que tem foco. Elementos focáveis podem variar entre os browsers, mas elementos de formulário podem sempre receber foco, então são candidatos para este tipo de evento.

Por exemplo, considere o HTML:

<form>
	<fieldset>
  	<input id="target" type="text" value="Hello there" />
	</fieldset>
</form>
<div id="other">
  Dispara o evento
</div>

O manipulador de evento pode ser acoplado ao seguinte campo input:

$('#target').keypress(function() {
  alert('Manipulador para .keypress() chamado.');
});

Agora quando o ponto de inserção está dentro do campo, pressionando uma tecla irá mostrar o alerta:

Manipulador para .keypress() chamado.

A mensagem se repete se a tecla é segurada. Para disparar o evento manualmente, aplique .keypress() sem um argumento:

$('#other').click(function() {
  $('#target').keypress();
});

Depois que este código executa, clica em Dispara o evento sempre alertará a mensagem.

Se a tecla pressiona qualquer lugar precisar ser capturada (por exemplo, para implementar teclas de atalho global em uma página), é útil anexar este comportamento para o objeto document. Causa da bolha de eventos, toda tecla pressionada fará seu caminho por todo o DOM até o objeto document a menos que seja explícitamente parada.

Para determinar qual caracter é entrado, examine o objeto event que é passado para a função do manipulador. Enquato browsers usam diferentes propriedades para armazenar esta informação, o jQuery normaliza a propriedade .which, então seguramente você pode usá-lo para receber o código do caractere.

Note que keydown e keyup promovem um indicativo de qual tecla é pressionada, enquando keypress indica qual caractere foi digitado. Por exemplo, um "a" minúsculo será reportado como 65 por keydown e keyup, mas como 97 por keypress. Um "A" maiúsulo é reportado como 65 para todos os eventos. Por causa desta distinção, quando capturado pulsações de tecla como setas, .keydown() ou .keyup() é uma melhor escolha.

<!DOCTYPE html>
<html>
<head>
  <style>
fieldset { margin-bottom: 1em; }
input { display: block; margin-bottom: .25em; }
#print-output {
  width: 100%;
}

  white-space: pre;
  padding: 5px;
  font-family: monaco, monospace;
  font-size: .7em;
}

</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>
  <fieldset>
    <label for="target">Type Something:</label>
    <input id="target" type="text" />
  </fieldset>
</form>
<button id="other">
  Dispara o evento
</button>
<script type="text/javascript" src="/scripts/events.js"></script>
<script>
var xTriggered = 0;
$('#target').keypress(function(event) {
  if (event.which == '13') {
     event.preventDefault();
   }
   xTriggered++;
   var msg = 'Manipulador para .keypress() chamado ' + xTriggered + ' vez(es).';
  $.print(msg, 'html');
  $.print(event);
});

$('#other').click(function() {
  $('#target').keypress();
});</script>

</body>
