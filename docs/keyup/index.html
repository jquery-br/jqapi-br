.keyup()

Categories: Events > Keyboard Events

.keyup( handler(eventObject) ) Retorna: jQuery

Descrição: Adiciona um manipulador ao evento Javascript "keyup" ou dispara esse evento para um elemento.

  • adicionado na versão: 1.0.keyup( handler(eventObject) )

    handler(eventObject)Uma função para executar a cada vez que o evento é disparado.

  • adicionado na versão: 1.4.3.keyup( [ eventData ], handler(eventObject) )

    eventDataUm mapa de dados que será passado ao manipulador de evento.

    handler(eventObject)Uma função a executar a cada vez que o evento é disparado.

  • adicinado na versão: 1.0.keyup()

Este método é um atalho para .bind('keyup', handler) nas duas primeiras variações, e .trigger('keyup') na terceira.

O evento keyup é enviado a um elemento quando o usuário solta uma tecla. Ele pode ser acoplado a um elemento, mas o evento é apenas enviado para o elemento que tem o foco. Elementos focáveis podem variar entre browsers, mas elementos de formulário pode sempre ter foco então são bons candidatos para este tipo de evento.

Por exemplo, considere o HTML:

<form>
  <input id="target" type="text" value="Hello there" />
</form>
<div id="other">
  Dispara o manipulador
</div>

O manipulador de evento pode ser acoplado ao campo input:

$('#target').keyup(function() {
  alert('Manipulador para .keyup() chamado.');
});

Agora quando o ponto de inserção é dentro do campo e uma tecla é pressionada e solta, o alerta é mostrado:

Manipulador para .keyup() chamado.

Para disparar o evento manualmente, aplique .keyup() sem argumentos:

$('#other').click(function() {
  $('#target').keyup();
});

Depois que este código executa, clica sobre Disparar o manipulador tambem alertará a mensagem.

Se a tecla pressionada em qualquer lugar precisa ser capturada (por exemplo, para implementar atalho global em uma página), é útil acoplar este comportamento ao objeto document. Por causa da bolha de evento, toda tecla pressionada fará seu caminho até o objeto DOM document a não ser que seja explícitamente parado.

Para determinar qual tecla foi pressionada, examine o objeto de evento que é passado à função manipuladora. Enquanto browsers usam diferentes propriedades para armazenar esta informação, jQuery normaliza a propriedade .which então você pode seguramente usá-lo para obter o código da tecla. Esse código corresponde a uma tecla do teclado, incluindo códigos para caracteres especiais como as setas. Para capturar o a entrada real de texto, .keypress() pode ser uma melhor escolha.

Mostra o objeto de evento pra o manipulador keyup quando uma tecla é solta no input.

<!DOCTYPE html>
<html>
<head>
  <style>
fieldset { margin-bottom: 1em; }
input { display: block; margin-bottom: .25em; }
#print-output {
  width: 100%;
}
.print-output-line {
  white-space: pre;
  padding: 5px;
  font-family: monaco, monospace;
  font-size: .7em;
}

</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>
  <fieldset>
    <label for="target">Tipo alguma coisa: </label>
    <input id="target" type="text" />
  </fieldset>
</form>
<button id="other">
  Dispara o manipulador
</button>
<script type="text/javascript" src="/scripts/events.js"></script>
<script>
var xTriggered = 0;
$('#target').keyup(function(event) {
  if (event.keyCode == '13') {
     event.preventDefault();
   }
   xTriggered++;
   var msg = 'Manipulador para .keyup() chamado ' + xTriggered + ' vez(es).';
  $.print(msg, 'html');
  $.print(event);
});

$('#other').click(function() {
  $('#target').keyup();
});</script>

</body>
</html>
