Seletor :lt()

Categorias: Seletores > Filtros básicos

Seletor lt

adicionado na versão: 1.0jQuery(':lt(index)')

  • index
    Index baseado no zero.

Descrição: Seleciona todos os elementos com o index menor que o passado na função.

Seletores relacionados com o index

Os seletores relacionados com o index (incluindo esse seletor "menor que") filtra os elementos baseado no seletor anterior. Eles limitam o conjunto com base na ordem dos elementos dentro do conjunto. Por exemplo, se elementos são primeiramente selecionados com o seletor de class (.myclass) e quatro elementos são retornados, esses elementos são dados do indice 0 até o 3.

Note que os arrays no JavaScript usam o index baseado no 0, esses seletores refletem esse efeito. É por isso que $('.myclass:lt(1)') seleciona o primeiro elemento no documento com a classe myclass ao invés de não selecionar nenhum. Em contraste :nth-child(n) usa index baseado em 1 em conformidade com as especificações do CSS

Example:

Encontra TDs com index menor que 4 (TD#4).

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <table border="1">

  <tr><td>TD #0</td><td>TD #1</td><td>TD #2</td></tr>
  <tr><td>TD #3</td><td>TD #4</td><td>TD #5</td></tr>

  <tr><td>TD #6</td><td>TD #7</td><td>TD #8</td></tr>
</table>
<script>$("td:lt(4)").css("color", "red");</script>

</body>
</html>
