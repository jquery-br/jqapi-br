.mousemove()

Categories: Events > Mouse Events

.mousemove( handler(eventObject) ) Retorna: jQuery

Descrição: Acopla um menipulador de evento para o evento JavaScript "mousemove" ou o dispara a um elemento.

  • adicionado na versão: 1.0.mousemove( handler(eventObject) )

    handler(eventObject)Uma função para executar cada vez que o evento é disparado.

  • adicionado na versão: 1.4.3.mousemove( [ eventData ], handler(eventObject) )

    eventDataUm mapa de dados que será passado ao manipulador de evento.

    handler(eventObject)Uma função para executar cada vez que o evento é disparado.

  • adicionado na versão: 1.0.mousemove()

Este método é um atalho para .bind('mousemove', handler) nas duas primeiras variações, e .trigger('mousemove') na terceira.

O evento mousemove é enviado a um elemento quando o pontreiro do mouse é movido para dentro do elemento. Qualquer elemento HTML pode receber este evento.

Por exemplo, considere o HTML:

<div id="target">
  Mova aqui
</div>
<div id="other">
  Dispara o manipulador
</div>
<div id="log"></div>

O manipulador de evento pode ser ligado ao alvo:

$("#target").mousemove(function(event) {
  var msg = "Manipulador para .mousemove() chamado em ";
  msg += event.pageX + ", " + event.pageY;
  $("#log").append("<div>" + msg + "</div>");
});

Agora quando o ponteiro do mouse se move dentro do botão alvo, as mensagens são colocadas no fim de <div id="log">:

Manipulador para .mousemove() chamado em (399, 48)
Manipulador para .mousemove() chamado em (398, 46)
Manipulador para .mousemove() chamado em (397, 44)
Manipulador para .mousemove() chamado em (396, 42)

Para disparar o evento manualmente, aplique .mousemove() sem um argumento:

$("#other").click(function() {
  $("#target").mousemove();
});

Depois que este código executa, cliques no botão Disparae também colocarão a seguinte mensagem ao final:

Manipulador para .mousemove() chamado em (undefined, undefined)

Quando acompanha o movimento do mouse, você normalmente precisa saber a posição real do ponteiro do mouse. O objeto de evento que é passado ao manipulador contem algumas informaçõoes sobre as coordenadas do mouse. Propriedades como .clientX, .offsetX, e .pageX estão disponíveis, mas o suporte a eles difere a entre os browsers. Felizmente, jQuery normaliza as propriedades .pageX e .pageY então eles podem ser usados em todos os browsers. Essas propriedades provêm as coordenadas X e Y do ponteiro do mouse relativos ao canto topo esquerdo do documento, como no exemplo de saída abaixo.

Mantenha em mente que o evento mousemove é disparado quando o ponteiro do mouse move, mesmo por um pixel. Isto significa que centenas de eventos podem ser gerados sobre uma quantia muito pequena de tempo. Se o manipulador tem que fazer muito processo significativo, ou se múltiplos manipuladores para o evento existir, isto pode ser um sério dreno de performance no browser. É iimportantante, portanto, otimizar manipuladores mousemove assim que possível, e para desacoplá-los assim que eles não serão mais necessários.

Um padrão comum é para acoplar o manipulador mousemove de dentro de um manipulador mousedown, e para desacoplá-lo de um manipulador correspondente mouseup. Se implementar essa sequência de eventos, lembre-se que o evento mouseup pode ser enviado a um elemento HTML diferente do evento mousemove. Para relatar isso, o manipulador mouseup deveria tipicamente ser acoplado a um elemento acima na árvore DOM, como <body>.

Exemplo:

Mostra as coordenadas do mouse quando o mouse é movido para a div amarela. Coorenadas são relativas à janela, que neste caso é o iframe.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width:220px; height:170px; margin;10px; margin-right:50px;
        background:yellow; border:2px groove; float:right; }
  p { margin:0; margin-left:10px; color:red; width:220px;
      height:120px; padding-top:70px;
      float:left; font-size:14px; }
  span { display:block; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>   
    Tente fazer scrolling também.
    <span>Mova o mouse sobre a div.</span>
    <span>&nbsp;</span>
  </p>

  <div></div>
<script>
    $("div").mousemove(function(e){
      var pageCoords = "( " + e.pageX + ", " + e.pageY + " )";
      var clientCoords = "( " + e.clientX + ", " + e.clientY + " )";
      $("span:first").text("( e.pageX, e.pageY ) - " + pageCoords);
      $("span:last").text("( e.clientX, e.clientY ) - " + clientCoords);
    });

</script>

</body>
</html>
