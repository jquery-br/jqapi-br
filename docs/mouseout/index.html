.mouseout()

Categories: Events > Mouse Events

.mouseout( handler(eventObject) ) Retorna: jQuery

Descrição: Acopla um elemento para o evento JavaScript "mouseout" ou dispara esse evento a um elemento.

  • adicionado na versão: 1.0.mouseout( handler(eventObject) )

    handler(eventObject)Uma função para executar a cada vez que o evento é disparado.

  • adicionado na versão: 1.4.3.mouseout( [ eventData ], handler(eventObject) )

    eventDataUm mapa de dados que será passado ao manipulador de evento.

    handler(eventObject)Uma função para executar a cada vez que o método é disparado.

  • adicionado na versão: 1.0.mouseout()

Este método é um atalho para .bind('mouseout', handler) nas duas primeiras variações, e .trigger('mouseout') na terceira.

O evento mouseout é enviado a um elemento quando o ponteiro do mouse deixa o elemento. Qualquer elemento HTML pode receber este evento.

Por exemplo, considere o HTML:

<div id="outer">
  Fora
  <div id="inner">
    Dentro
  </div>
</div>
<div id="other">
  Dispara o manipulador
</div>
<div id="log"></div>

O manipulador pode ser ligado a qualquer elemento:

$('#outer').mouseout(function() {
  $('#log').append('Manipulador para .mouseout() chamado.');
});

Agora quando o ponteiro do mouse move para fora da <div> Fora, a mensagem é colocada no final de <div id="log">. Para disparar o evento manualmente, aplique .mouseout() sem um argumento:

$('#other').click(function() {
  $('#outer').mouseout();
});

Depois que este código executa, cliques sobre Disparar o manipulador também colocarão no final da mensagem.

O tipo de evento pode causar muitas dores de cabeça devido ao bubbling de evento. Por exemplo, quando o ponteiro do mouse move para fora do elemento Dentro neste exemplo, um evento mouseout será enviado para ele, então chegar até o Fora. Isto pode disparar a o manipulador ligado mouseout em horas inoportunas. Veja a discussão oara .mouseleave() para uma alternativa útil.

Exemplo:

Mostra o número de vezes que os eventos mouseout e mouseleave são disparados. mouseout dispara quando o ponteiro sai do elemento filho também, enquanto mouseleave dispara apenas quando o ponteiro sai do elemento acoplado.

<!DOCTYPE html>
<html>
<head>
  <style>
div.out {
width:40%;
height:120px;
margin:0 15px;
background-color:#D6EDFC;
float:left;
}
div.in {
width:60%;
height:60%;
background-color:#FFCC00;
margin:10px auto;
}
p {
line-height:1em;
margin:0;
padding:0;
}
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div class="out overout"><p>move your mouse</p><div class="in overout"><p>move your mouse</p><p>0</p></div><p>0</p></div>

<div class="out enterleave"><p>move your mouse</p><div class="in enterleave"><p>move your mouse</p><p>0</p></div><p>0</p></div>


<script>
    var i = 0;
    $("div.overout").mouseout(function(){
      $("p:first",this).text("mouse out");
      $("p:last",this).text(++i);
    }).mouseover(function(){
      $("p:first",this).text("mouse over");
    });

    var n = 0;
    $("div.enterleave").bind("mouseenter",function(){
      $("p:first",this).text("mouse enter");
    }).bind("mouseleave",function(){
      $("p:first",this).text("mouse leave");
      $("p:last",this).text(++n);
    });

</script>

</body>
</html>
