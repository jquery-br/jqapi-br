.mouseover()

Categories: Events > Mouse Events

.mouseover( handler(eventObject) ) Retorna: jQuery

Descrição: Acopla um manipulador de evento para o evento JavaScript "mouseover" JavaScript, ou dispara aquele evento a um elemento.

  • adicionado na versão: 1.0.mouseover( handler(eventObject) )

    handler(eventObject) Uma função para executar cada vez que o evento é disparado.

  • adicionado na versão: 1.4.3.mouseover( [ eventData ], handler(eventObject) )

    eventData Um mapa de dados que será passado a um manipulador de evento.

    handler(eventObject) Uma função para executar cada vez que o evento é disparado.

  • adicionado na versão: 1.0.mouseover()

Este método é um atalho para .bind('mouseover', handler) nas duas primeiras variações, e and .trigger('mouseover') na terceira.

O evento mouseover é enviado a um elemento quando o ponteiro do mouse passa por cima do elemento. Qualquer elemento HTML pode receber este evento.

Por exemplo, considere o HTML:

<div id="outer">
  Fora
  <div id="inner">
    Dentro
  </div>
</div>
<div id="other">
  Dispara o manipulador
</div>
<div id="log"></div>

O manipulador de evento pode ser ligado a qualquer elemento:

$('#outer').mouseover(function() {
  $('#log').append('<div>Manipulador para .mouseover() chamado.</div>');
});

Agora quando o ponteiro do mouse move sobre a <div> Fora, a mensagem é colocada no final de <div id="log">. Nós podemos também disparar o evento quando um outro elemento é clicado:

$('#other').click(function() {
  $('#outer').mouseover();
});

Depois que este código executa, cliques sobre Disparar o manipulador também colocarão a mensagem no final.

Este tipo de evento pode causar muitas dores de cabeça devido ao bubbluing de evento. Por exemplo, quando o ponteiro do mouse se move sobre o elemento Dentro neste exemplo, um evento mouseover será mandado para ele, entao chega até o Fora. Isto pode duisparar nosso manipulador acoplado mouseover em horas inoportunas. Veja a discussão para .mouseenter() para uma anternativa útil.

Exemplo:

Mostra o número de vezes que os eventos mouseover e mouseenter são disparados. mouseover dispara quando o ponteiro se move sobre um elemento filho também, enquanto mouseenter dispara apenas quando o ponteiro se move sobre o elemento acoplado.

<!DOCTYPE html>
<html>
<head>
  <style>
div.out { width:40%; height:120px; margin:0 15px;
          background-color:#D6EDFC; float:left; }
div.in {  width:60%; height:60%; 
          background-color:#FFCC00; margin:10px auto; }
p { line-height:1em; margin:0; padding:0; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div class="out overout">
  <span>mova seu mouse</span>
  <div class="in">
  </div>
</div>

<div class="out enterleave">
  <span>mova seu mouse</span>
  <div class="in">
  </div>
</div>

<script>
  var i = 0;
  $("div.overout").mouseover(function() {
    i += 1;
    $(this).find("span").text( "mouse over x " + i );
  }).mouseout(function(){
    $(this).find("span").text("mouse out ");
  });

  var n = 0;
  $("div.enterleave").mouseenter(function() {
    n += 1;
    $(this).find("span").text( "mouse enter x " + n );
  }).mouseleave(function() {
    $(this).find("span").text("mouse leave");
  });

</script>

</body>
</html>
