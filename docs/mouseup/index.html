Este método é um atalho para .bind('mouseup', handler) na primeira variação, e .trigger('mouseup') na segunda.

O evenyo mouseup é enviado a um elemento quando o ponteiro do mouse está sobre o elemento e o ponteiro do mouse for solto. Qualquer elemento HTML pode receber este evento.

Por exemplo, considere o HTML:

<div id="target">
  Clique aqui
</div>
<div id="other">
  Manipula o evento
</div>

O manipulador de evento pode ser acoplado a qualquer <div>:

$('#target').mouseup(function() {
  alert('Manipulador para .mouseup() chamado.');
});

Agora se nós clicarmos sobre este elemento, o alert é mostrado:

Manipulador para .mouseup() chamado.

Podemos também dispara o evento quando um elemtno diferente é clicado:

$('#other').click(function() {
  $('#target').mouseup();
});

Depois que isso executa, cliques sobre Dispara o manipulador tambéo mostrarão o alert da mensagem.

Se o usuario clicar fora de um elemento, arrastar e soltar o botão, isto ainda conta como um evento mouseup. Exemplo:

Mostra textos quando disparados os eventos mouseup e mousedown.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Pressione o mouse e solte aqui.</p>

<script>
    $("p").mouseup(function(){
      $(this).append('<span style="color:#F00;">Mouse up.</span>');
    }).mousedown(function(){
      $(this).append('<span style="color:#00F;">Mouse down.</span>');
    });

</script>

</body>
</html>
