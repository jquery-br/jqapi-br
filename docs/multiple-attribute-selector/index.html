Multiple Attribute Selector [name="value"][name2="value2"]

Categorias: Selectors > Attribute

attributeMultiple selector

adicionado na versão: 1.0jQuery('[attributeFilter1][attributeFilter2][attributeFilterN]')

  • attributeFilter1
    Atributo filtro.
    attributeFilter2
    Outro atributo filtro, reduzindo ainda mais a seleção
    attributeFilterN
    Quantos atributos filtros mais forem necessários

Descrição: Encontra elementos que correspodem a todos os atributos filtros especificados.

Exemplo:

Encontra todos inputs que tem um atributo id e que o atributo nome termine com 'man' e atribui o valor.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <input id="man-news" name="man-news" />

  <input name="milkman" />
  <input id="letterman" name="new-letterman" />
  <input name="newmilk" />
<script>$('input[id][name$="man"]').val('only this one');</script>

</body>
</html>
