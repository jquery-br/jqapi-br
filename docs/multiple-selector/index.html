Multiple Selector (“selector1, selector2, selectorN”)

Categorias: Selectors > Basic

multiple selector

adicionado na versão: 1.0jQuery('selector1, selector2, selectorN')

  • selector1
    Qualquer seletor válido.
    selector2
    Outro seletor válido.
    selectorN
    Quantos seletores mais que você queira.

Descrição: Seleciona os resultados combinados de todos os seletores especificados.

Você pode especificar qualquer número de seletores para combinar em um único resultado. Este combinador de expressão múltipla é um eficiente modo de selecionar diversos elementos. A ordem dos elementos DOM no retorno do objeto jQuery pode não ser idêntico, como eles estarão na ordem do documento. Uma alternativa a isto é o método .add().

Exemplos:

Exemplo: Encontra os elementos que correspondem a qualquer destes três seletores. 

<!DOCTYPE html>
<html>
<head>
  <style>

  div,span,p {
    width: 126px;
    height: 60px;
    float:left;
    padding: 3px;
    margin: 2px;
    background-color: #EEEEEE;
    font-size:14px;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>div</div>

  <p class="myClass">p class="myClass"</p>
  <p class="notMyClass">p class="notMyClass"</p>
  <span>span</span>
<script>$("div,span,p.myClass").css("border","3px solid red");</script>

</body>
</html>

Exemplo: Mostra a ordem no objeto jQuery. 

<!DOCTYPE html>
<html>
<head>
  <style>
  b { color:red; font-size:16px; display:block; clear:left; }
  div,span,p { width: 40px; height: 40px; float:left;
               margin: 10px; background-color: blue; 
               padding:3px; color:white; 
             }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <span>span</span>

  <p>p</p>
  <p>p</p>
  <div>div</div>
  <span>span</span>

  <p>p</p>
  <div>div</div>
  <b></b>
<script>
    var list = $("div,p,span").map(function () {
      return this.tagName;
    }).get().join(", ");
    $("b").append(document.createTextNode(list));
</script>

</body>
</html>
