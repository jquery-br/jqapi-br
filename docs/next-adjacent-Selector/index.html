Seletor Adjacente (“prev + next”)

Categorias: Selectors > Hierarchy

Seletor Adjacente

Adicionado na versão: 1.0jQuery('prev + next')

  • prev
    Qualquer seletor válido.
    next
    Seletor para selecionar o elemento que está ao lado do primeiro seletor.

Descrição: Seleciona todos os elementos que se encaixam em "next" que são imediatamente precendidos do irmãos "prev".

Um ponto importante a ser considerado com ambos, o seletor do irmão adjacente (prev + next) e o seletor geral de irmãos (prev ~ siblings) é que os elementos em ambos os lados da combinação devem dividir o mesmo pai..

Exemplo:

Encontra todos os inputs que são próximos ao label.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>

    <label>Name:</label>
    <input name="name" />
    <fieldset>
      <label>Newsletter:</label>

      <input name="newsletter" />
    </fieldset>
  </form>
  <input name="none" />
<script>$("label + input").css("color", "blue").val("Labeled!")</script>

</body>
</html>
