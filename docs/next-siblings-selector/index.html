Seletor de irmãos próximos (“prev ~ siblings”)

Categories: Selectors > Hierarchy

Seletor irmãos próximos

adicionado na versão: 1.0jQuery('prev ~ siblings')

  • prev
    Qualquer seletor válido.
    siblings
    Seletor para filtrar os elementos seguintes ao primeiro seletor.

Descrição: Seleciona todos os elementos irmãos que seguem após o elemento "prev", tem o mesmo pai, e se enquadram no seletor filtro "siblings".

A notavel diferença entre (prev + next) e (prev ~ siblings) são os seus alcances. Enquanto o anterior alcança apenas os irmãos imediatos, o segundo consegue alcançar todos os elementos irmão.

Exemplo:

Encontra todas as divs que são irmão após o elemento com #prev como id. Note que o span não é selecionado já que ele não é uma div e o "niece" é filho de um irmão e não realmente um irmão..

<!DOCTYPE html>
<html>

  <style>

    display:block;
    width:80px;
    height:80px;

    float:left;
    font-size:14px;
  }
  div#small {
    width:60px;
    height:25px;
    font-size:12px;
    background:#fab;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>div (doesn't match since before #prev)</div>
  <span id="prev">span#prev</span>
  <div>div sibling</div>

  <div>div sibling <div id="small">div niece</div></div>
  <span>span sibling (not div)</span>
  <div>div sibling</div>
<script>$("#prev ~ div").css("border", "3px groove blue");</script>

</body>
</html>
