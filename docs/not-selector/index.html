Seletor :not()

Categorias: Selectors > Basic Filter

not selector

adicionado na versão: 1.0jQuery(':not(selector)')

  • selector
    Um seletor para filtrar.

Descrição: Seleciona todos os elementos que não corresponde ao seletor.

Todos os seletores são aceitos pelo :not(), por exemplo: :not(div a) e :not(div,a).

Notas adicionais

O método .not() irá terminar fornecendo mais seletores legíveis que colocam seletores complexos ou variáveis dentro do filtro do seletor :not(). Na maioria dos casos, essa é a melhor escolha.

Exemplo:

Encontra todos os inputs que não estão checados e destaca o próximo irmão span. Observe que não há mudanças quando os checkboxes são clicados, pois nenhum evento ckick foi adicionado.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>
  <input type="checkbox" name="a" />
  <span>Mary</span>
</div>

<div>
  <input type="checkbox" name="b" />
  <span>lcm</span>

</div>
<div>
  <input type="checkbox" name="c" checked="checked" />

  <span>Peter</span>
</div>
<script>
  $("input:not(:checked) + span").css("background-color", "yellow");
  $("input").attr("disabled", "disabled");

</script>

</body>
</html>
