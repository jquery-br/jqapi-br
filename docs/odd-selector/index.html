Seletor :odd

Categorias: Selectors > Basic Filter

Seletor odd

adicionado na versão: 1.0jQuery(':odd')

Descrição: Seleciona elementos ímpares, baseado no index 0, Veja também even.

Em particular, note que baseado no inxex 0 significa, que o:even seleciona o segundo elemento, quarto e assim por diante.

Exemplo:

Procura por linhas pares na tabela, marcando o segundo, quarto e assim por diante (index 1, 3, 5 etc.).

<!DOCTYPE html>
<html>
<head>
  <style>

  table {
    background:#f3f7f5;
  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <table border="1">
    <tr><td>Linha com index #0</td></tr>
    <tr><td>Linha com index #1</td></tr>

    <tr><td>Linha com index #2</td></tr>
    <tr><td>Linha com index #3</td></tr>
  </table>
<script>$("tr:odd").css("background-color", "#bbbbff");</script>

</body>
</html>
