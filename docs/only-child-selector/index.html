:only-child Selector

Categorias: Selectors > Child Filter

only-child selector

adicionado na versão: 1.1.4jQuery(':only-child')

Descrição: Seleciona todos os elementos que são filhos únicos.

Se o pai tem outros elementos filhos, nada será selecionado.

Example:

Muda o texto e adiciona uma borda para cada botão que é filho único.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width:100px; height:80px; margin:5px; float:left; background:#b9e }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>
  <button>Irmão!</button>
  <button>Irmão!</button>
</div>

<div>
  <button>Irmão!</button>
</div>
<div>
  None
</div>

<div>
  <button>Irmão!</button>
  <button>Irmão!</button>
  <button>Irmão!</button>

</div>
<div>
  <button>Irmão!</button>
</div>
<script>
  $("div button:only-child").text("Sozinho").css("border", "2px blue solid");
</script>

</body>
</html>
