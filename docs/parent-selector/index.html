:parent Selector

Categorias: Selectors > Content Filter

seletor parent

adicionado na versão: 1.0jQuery(':parent')

Descrição: Seleciona todos os elementos que são os pais de um outro elemento, incluindo os nós de texto.

Este é o inverso de :empty.

Uma coisa importante a notar em relação ao uso de :parent (e :empty) é que os elemtnos filhos incluem nós de texto.

A W3C recomenda que o elemento <p> tenha no mínimo um nó filho, mesmo se o filho seja meramente texto (veja http://www.w3.org/TR/html401/struct/text.html#edef-P). Alguns outros elementos, por outro lado, são vazios (ex.: não tem filhos) por definição: <input>, <img>, <br>, e <hr>, por exemplo.

Exemplo:

Encontra todos os tds com filhos, incluindo texto.

<!DOCTYPE html>
<html>
<head>
  <style>
  td { width:40px; background:green; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <table border="1">

  <tr><td>Value 1</td><td></td></tr>
  <tr><td>Value 2</td><td></td></tr>

</table>
<script>$("td:parent").fadeTo(1500, 0.3);</script>

</body>
</html>
