:password Selector

Categorias: Selectors > Form

seletor password

adicionado na versão : 1.0jQuery(':password')

Descrição: Seleciona todos os elementos do tipo password.

$(':password') é o equivalente a $('[type=password]'). Como com outros seleres de pseudo-classe (aqueles que começam com um ":") é recomendado precedê-lo com uma tag name ou algum outro seletor; senão, o seletor universal ("*") é implicado. Em outras palavras, o mero $(':password') é o equivalente a $('*:password'), então $('input:password') devem ser usados ao invés disso.

Exemplo:

Encontra todos os inputs password.

<!DOCTYPE html>
<html>
<head>
  <style>
  textarea { height:45px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <form>
    <input type="button" value="Input Button"/>
    <input type="checkbox" />

    <input type="file" />
    <input type="hidden" />
    <input type="image" />

    <input type="password" />
    <input type="radio" />
    <input type="reset" />

    <input type="submit" />
    <input type="text" />
    <select><option>Option<option/></select>

    <textarea></textarea>
    <button>Button</button>
  </form>
  <div>
  </div>
<script>

    var input = $("input:password").css({background:"yellow", border:"3px red solid"});
    $("div").text("Para este tipo, jQuery encontrou " + input.length + ".")
            .css("color", "red");
    $("form").submit(function () { return false; }); // então ele não deve enviar

</script>

</body>
</html>
