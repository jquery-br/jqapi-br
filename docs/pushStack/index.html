.pushStack()

Categorias: Internals

.pushStack( elements ) Retorna: jQuery

Descrição: Adiciona uma coleção de elementos DOM na pilha jQuery.

  • adicionado na versão: 1.0.pushStack( elements )

    elementsUm array de elementos para colocar na pilha e transformar em um novo objeto jQuery.

  • adicionado na versão: 1.3.pushStack( elements, name, arguments )

    elementsUm array de elementos para colocar na pilha e transformar em um novo objeto jQuery.

    nameO nome de um método jQuery que gerou o array de elementos.

    argumentsOs argumentos que foram passandos ao método jQuery (por serialização).

Exemplo:

Adiciona alguns elementos na pilha jQuery, depois retira novamente.

jQuery([])
    .pushStack( document.getElementsByTagName("div") )
        .remove()
    .end();
