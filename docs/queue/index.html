.queue( [ nome_da_fila ] ) Retornar para: Array

Descrição: Mostra a fila de funções que estão para ser executadas dos elementos encontrados.

  • adicionado na versão: 1.2 .queue( [ nome_da_fila ] )

    nome_da_fila: Uma string contendo o nome da fila. O valor padrão é fx, esta é a fila padrão de efeitos que estão sendo executados.

Exemplo:

Mostra o tamanho da fila.

      $("div").show("slow");
      $("div").animate({left:'+=200'},2000);

.queue( [ nome_da_fila ], nova_fila ) Retornar para: jQuery

Descrição: Manipula a fila de funções que estão para ser executadas dos elementos encontrados.

  • adicionado na versão: 1.2 .queue( [ nome_da_fila ], nova_fila )

    nome_da_fila: Uma string contendo o nome da fila. O valor padrão é fx, esta é a fila padrão de efeitos que estão sendo executados.

    nova_fila: Um array de funções para substituir os itens que estão na fila.

  • adicionado na versão: 1.2 .queue( [ nome_da_fila ], função_retorno( próximo ) )

    nome_da_fila: Uma string contendo o nome da fila. O valor padrão é fx, esta é a fila padrão de efeitos que estão sendo executados.

    função_retorno(próximo): Uma nova função para ser adicionada para fila, com uma função para ser chamada que retirará o próximo item da fila.

Todo elemento pode ter uma ou mais filas de funções referênciadas pelo jQuery. Na maioria das aplicações somente uma fila (fx) é utilizada. A função queue permite que uma sequência de ações sejam chamadas em um elemento assíncronamente sem interromper a execução do programa. Um exemplo típico desta situação é a chamada de multíplas funções de animações em um elemento. Por exemplo:

$('#foo').slideUp().fadeIn();

Quando este trecho de código é executado, o elemento começa a sua animação (slideUp) imediatamente, mas a animação (fadeIn) é colocada na fila fx para ser chamada somente quando a animação (slideUp) estiver completa.

A função .queue() nos permite manipular diretamente a fila de funções. Chamando a função .queue() com uma função de retorno é particularmente vantajoso; ela nos permite adicionar uma nova função ao final da fila.

Este recurso é similar a quando fornecemos uma função de retorno para uma função de animação, mas não necessita que a função de retorno seja passada assim que a animação é executada.

$('#foo').slideUp();
$('#foo').queue(function() {
  alert('Animation complete.');
  $(this).dequeue();
});

Isto é equivalente a:

$('#foo').slideUp(function() {
  alert('Animation complete.');
});

Note que quando a função é adicionada utilizando a função .queue(), nós devemos garantir que eventualmente a função .dequeue() é chamada para que a próxima função na fila seja executada.

No jQuery 1.4 a função que é chamada é passada em outra função como o primeiro argumento, que quando chamado automaticamente executa a função dequeue para o próximo item e mantém a fila em movimento. Você deverá utilizar desta forma:

$("#test").queue(function(next) {
    // Do some stuff...
    next();
});

Exemplos:

Exemplo: Adiciona na fila uma função personalizada. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:3px; width:40px; height:40px;
        position:absolute; left:0px; top:30px; 
        background:green; display:none; }
  div.newcolor { background:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Click here...
  <div></div>
<script>$(document.body).click(function () {
      $("div").show("slow");
      $("div").animate({left:'+=200'},2000);
      $("div").queue(function () {
        $(this).addClass("newcolor");
        $(this).dequeue();
      });
      $("div").animate({left:'-=200'},500);
      $("div").queue(function () {
        $(this).removeClass("newcolor");
        $(this).dequeue();
      });
      $("div").slideUp();
    });</script>

</body>
</html>

Exemplo: Adiciona um array de filas para substituir a fila atual. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:3px; width:40px; height:40px;
        position:absolute; left:0px; top:30px; 
        background:green; display:none; }
  div.newcolor { background:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="start">Start</button>
  <button id="stop">Stop</button>
  <div></div>
<script>$("#start").click(function () {
      $("div").show("slow");
      $("div").animate({left:'+=200'},5000);
      $("div").queue(function () {
        $(this).addClass("newcolor");
        $(this).dequeue();
      });
      $("div").animate({left:'-=200'},1500);
      $("div").queue(function () {
        $(this).removeClass("newcolor");
        $(this).dequeue();
      });
      $("div").slideUp();
    });
    $("#stop").click(function () {
      $("div").queue("fx", []);
      $("div").stop();
    });</script>

</body>
</html>
