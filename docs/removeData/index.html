.removeData()

Categorias: Data | Miscellaneous > Data Storage

.removeData( [ name ] ) Retorna: jQuery

Descrição: Remove um pedaço de dados previamente guardado.

  • adicionado na versão: 1.2.3.removeData( [ name ] )

    nameUma string nomeando o pedaço dos dados para deletar.

O método .removeData() nos permite remover valores que foram previamente setados usando .data(). Quando chamamos com o nome da chave, o .removeData() deleta esse valor em particular; quando chamamos sem argumentos, todos os valores são removidos.

NOTA: Começando com o jQuery 1.4.3, ao chamar .removeData() vai causar a remoção do valor da propriedade para reverter ao valor do dado do atributo com o mesmo nome na DOM, ao invés de ser setado como undefined.

Exemplo:

Definir dados armazenados para 2 nomes e após remover um deles.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { margin:2px; color:blue; }
  span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>value1 antes da criação: <span></span></div>
  <div>value1 depois da criação: <span></span></div>
  <div>value1 após remoção: <span></span></div>

  <div>value2 após remoção: <span></span></div>
<script>

    $("span:eq(0)").text("" + $("div").data("test1"));
    $("div").data("test1", "VALUE-1");
    $("div").data("test2", "VALUE-2");
    $("span:eq(1)").text("" + $("div").data("test1"));
    $("div").removeData("test1");
    $("span:eq(2)").text("" + $("div").data("test1"));
    $("span:eq(3)").text("" + $("div").data("test2"));

</script>

</body>
</html>
