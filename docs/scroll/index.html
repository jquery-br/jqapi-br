.scroll()

Categoria: Events > Browser Events

.scroll( handler(eventObject) ) Retorna: jQuery

Descrição: Adiciona um handler ao evento "scroll" do JavaScript ou dispara o evento em um elemento.

  • adicionado na versão: 1.0.scroll( handler(eventObject) )

    handler(eventObject)Uma função a ser executada cada vez que o evento é disparado.

  • adicionado na versão: 1.4.3.scroll( [ eventData ], handler(eventObject) )

    eventDataUm mapa de dados que será passado ao handler.

    handler(eventObject)Uma função a ser executada cada vez que o evento é disparado.

  • adicionado na versão: 1.0.scroll()

Esse método é um atalho para .bind('scroll', handler) na priemeira e segunda variação, e .trigger('scroll') na terceira.

O evento scroll é enviado a um elemento quando o usuário rolar a um lugar diferente no elemento. Aplica-se ao objeto window, mas também à frames roláveis e elementos com a propriedade CSS overflow definida como scroll (ou auto quando o elemento explícita a altura, ou a largura é menor que a altura ou largura desse conteúdo).

<div id="target" style="overflow: scroll; width: 200px; height: 100px;">

<div id="other">

<div id="log"></div>

O estilo definido é para deixar o elemento alvo menor o suficiente para ser rolável:

O handler do scroll pode ser ligado nesse elemento:

$('#target').scroll(function() {
  $('#log').append('<div>Handler do .scroll() chamado.</div>');
});

Agora quando o usuário rolar o texto para cima ou para baixo, uma ou mais mensagens irão aparecer na <div id="log"></div>:

Handler para .scroll() chamado.

Para disparar o evento manualmente, aplicar .scroll() sem argumentos:

$('#other').click(function() {
  $('#target').scroll();
});

Após esse código ser executado, ao clicar em "Dispare o handler" irá adicionar a mensagem.

Um evento scroll é enviado sempre que a posição do scroll de um elemento muda, independentemente da causa. Um click, ou ao arrastar a barra de rolagem, arrastando dentro do elemento, apertando as flechinhas, ou usando a rodinha do mouse, pode causar o evento.
