Seletor :selected

Categorias: Seletores > Formulários

Seletor selected

adicionado na versão: 1.0jQuery(':selected')

Descrição: Seleciona todos os elementos que estão selecionados.

O seletor :selected trabalha com elementos <option>. Não funciona para checkboxes ou inputs do tipo radio; use :checked para eles.

Exemplo:

Adiciona um evento ao select para pegar o texto de cada option selecionado e escreve eles em uma div. O código também inicialmente dispara o evento para a escrita do texto quando a página for carregada.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <select name="garden" multiple="multiple">

    <option>Flowers</option>
    <option selected="selected">Shrubs</option>
    <option>Trees</option>
    <option selected="selected">Bushes</option>

    <option>Grass</option>
    <option>Dirt</option>
  </select>
  <div></div>
<script>

    $("select").change(function () {
          var str = "";
          $("select option:selected").each(function () {
                str += $(this).text() + " ";
              });
          $("div").text(str);
        })
        .trigger('change');
</script>

</body>
</html>
