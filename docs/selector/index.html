.selector

Categorias: Internals | Properties > Propriedades de Instancias de Objetos jQuery

selector Retorna: String

Descrição: Um seletor representando o seletor originalmente passado ao jQuery().

  • adicionado na versão: 1.3selector

Deveria ser usado em conjunção a um contexto para determinar a query exata usada.

O método .live() para manipulação de eventos de ligação usa esta propriedade para determinar como realizar suas pesquisas. Plug-ins que realizam tarefas similares podem também considerar a propriedade útil. Esta propriedade contém uma string representando o conjunto combinado de elementos, mas se os métodos transversais DOM já foram chamados no objeto, a string pode não ser uma expressão de seletor jQuery válida. Por essa razão, o valor do .selector é geramente mais útil imediatamente seguindo a criação original do objeto. Consequentemente, o método .live() só deve ser utilizado neste cenário.

Exemplos:

Exemplo: Determina o seletor usado. 

<!DOCTYPE html>
<html>
<head>
  <style>
  body { cursor:pointer; }
  div { width:50px; height:30px; margin:5px; float:left;
        background:green; }
  span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Some selectors:<ul></ul>
<script>$("ul")
  .append("<li>" + $("ul").selector + "</li>")
  .append("<li>" + $("ul li").selector + "</li>")
  .append("<li>" + $("div#foo ul:not([class])").selector + "</li>");

</script>

</body>
</html>

Exemplo: Coletando elementos diferentemente 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Some selectors:<ul></ul>

<script>
   $('<div>' + $('ul li.foo').selector + '</div>').appendTo('body');  // "ul li.foo"
   $('<div>' + $('ul').find('li').filter('.foo').selector + '</div>').appendTo('body'); // "ul li.filter(.foo)"
</script>

</body>
</html>
