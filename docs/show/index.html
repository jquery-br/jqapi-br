.show()

Categoria: Efeitos > Básicos

.show( ) Retornar para: jQuery

Descrição: Motra os elementos encontrados.

  • adicionado na versão: 1.0 .show()

  • adicionado na versão: 1.0 .show( duração, [ função_de_retorno ] )

    duração: Uma string ou número determinando a duração da animação.

    função_de_retorno: Uma função a ser chamada quando a animação estiver completa.

  • adicionado na versão: 1.4.3.show( [ duração ], [ função_de_efeito ], [ função_de_retorno ] )

    duração: Uma string ou número determinando a duração da animação.

    função_de_efeito: Uma string indicando qual função de animação utilizar durante a transição.

    função_de_retorno: Uma função a ser chamada quando a animação estiver completa.

Sem parâmetros, a função .show() é o jeito mais simples de visualizar um elemento: 

$('.target').show();

Os elementos encontrados serão revelado imediatamente sem um efeito de transição (animação). Isto é equivalente a chamar o método .css('display', 'block'), exceto que a propriedade display é restaurada ao que era inicialmente. Quando um elemento possui a propriedade display com o valor inline e é escondido (hidden) e mostrada (show) ela será novamente visualizada com a propriedade com o valor inline.

Nota: Se estiver utilizando o valor !important no estilos, por exemplo: display: none !important, é necessário sobrescrever o estilo (style) utilizando .css('display', 'block !important') para utilizar a função .show() corretamente.

Quando a duração é fornecida, a função .show() se torna uma função de animação. A função .show() anima as propriedades width, height, e opacity dos elementos encontrados simultâneamentes.

O valor da duração são fornecidos utilizando milisegundos; valores maiores indicam animação mais lentas e não mais rápidas. As strings 'fast' e 'slow' podem ser fornecidas para indicar a duração de 200 e 600 milisegundos respectivamente.

Com a versão do jQuery 1.4.3, uma string opcional (função_de_efeito) pode ser utilizada. Estas funções de efeitos especificam a velocidade que a animação procede em diferentes pontos da animação. A função padrão implementada no jQuery é a swing, e outra que progride constantemente chamada linear. Mais funções de animações podem ser utilizadas através de plug-ins como o jQuery UI.

Se fornecida, a função de retorno é ativada assim que a animação estiver completa. Isto pode ser útil para encadear diferentes animações em sequência. A função de retorno não utiliza nenhum parâmetro, porém pode ser utilizado a referência this para obter o elemento DOM que está sendo animado. Se múltiplos elementos estão sendo animados, é importante ressaltar que a função de retorno será chamada para cada elemento animado, e não uma vez para a animação com um todo.

Nós podemos animar qualquer elemento asism como uma simples imagem:

<div id="clickme">
  Click here
</div>
<img id="book" src="book.png" alt="" width="100" height="123" />
With the element initially hidden, we can show it slowly:
$('#clickme').click(function() {
  $('#book').show('slow', function() {
    // Animation complete.
  });
});

Notas Adicionais:

  • Todos os efeitos do jQuery, incluindo .show(), podem ser desativados configurando o valor globalmente utilizando o código: jQuery.fx.off = true, o qual efetivamente muda a duração do efeito para 0. Para maiores informações, veja jQuery.fx.off.

Exemplos:

Exemplo: Anima todos os parágrafos mostrando-os lentamente (slow), completando a animação em 600 milisegundos. 

<!DOCTYPE html>
<html>
<head>
  <style>
      p { background:yellow; }
      </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Show it</button>

      <p style="display: none">Hello  2</p>
<script>
    $("button").click(function () {
    $("p").show("slow");
    });
    </script>

</body>
</html>

Exemplo: Anima todos as DIVs ocultas mostrando-as rapidamente em ordem, completando cada animação em 200 milisegundos. Assim que cada animação estiver completa, ele inicia a próxima. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background:#def3ca; margin:3px; width:80px; 
  display:none; float:left; text-align:center; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <button id="showr">Show</button>
  <button id="hidr">Hide</button>
  <div>Hello 3,</div>

  <div>how</div>
  <div>are</div>
  <div>you?</div>
<script>
$("#showr").click(function () {
  $("div:eq(0)").show("fast", function () {
    /* use callee so don't have to name the function */
    $(this).next("div").show("fast", arguments.callee);
  });
});
$("#hidr").click(function () {
  $("div").hide(2000);
});

</script>

</body>
</html>

Exemplo: Mostra todos os elementos SPAN e INPUT utilizando uma animação. Assim que uma animação estiver completa, ele muda para a próxima. 

<!DOCTYPE html>
<html>
<head>
  <style>
  span { display:none; }
  div { display:none; }
  p { font-weight:bold; background-color:#fcd; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Do it!</button>
  <span>Are you sure? (type 'yes' if you are) </span>
  <div>
    <form>
      <input type="text"  value="as;ldkfjalsdf"/>
    </form>
  </div>
  <p style="display:none;">I'm hidden...</p>
  
<script>
function doIt() {
  $("span,div").show("slow");
}
/* can pass in function name */
$("button").click(doIt);

$("form").submit(function () {
  if ($("input").val() == "yes") {
    $("p").show(4000, function () {
      $(this).text("Ok, DONE! (now showing)");
    });
  }
  $("span,div").hide("fast");
  /* to stop the submit */
  return false; 
});
</script>

</body>
</html>
