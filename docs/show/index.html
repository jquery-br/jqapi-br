.show()

Categories: Effects > Basics

.show( ) Retorna: jQuery

Descrição: Exibe os elementos correspondentes.

  • adicionado na versão: 1.0.show()

  • adicionado na versão: 1.0.show( duration, [ callback ] )

    durationUma palavra ou número que determina por quanto tempo durará a animação.

    callbackUma função para ser executada quando a animação finalizar.

  • adicionado na versão: 1.4.3.show( [ duration ], [ easing ], [ callback ] )

    durationUma palavra ou número que determina por quanto tempo durará a animação.

    easingUma palavra que determina qual função de suavização será usada para a transição.

    callbackUma função para ser executada quando a animação finalizar.

Quando não houver parâmetros no método .show(), simplesmente é mostrado o elemento na tela: 

$('.target').show();

O elemento correspondente será revelado imediatamento, sem animação alguma. Isto é basicamente equivalente a chamar .css('display', 'block'), com a exceção que a propriedade display é restaurada para o que era inicialmente. Se um elemento tem a propriedade display com valor inline, então esconde e mostra o elemento, mais uma vez com o valor inline.

Observação: Se estiver usando !important no seu CSS, como por exemplo: display: none !important, É necessário sobrescrever o CSS usando .css('display', 'block !important'), se quizer que o método .show() funcione corretamente.

Quando a duration(duração) está explicita, o método .show() se torna uma animação. O método .show() anima a largura(width), altura(height), e opacidade(opacity) do elemento correspondente simultaneamente.

A duração é explicita em milisegundos; Valores maiores indicam animações mais demoradas, não mais rápidas. As palavras 'fast' e 'slow' podem ser fornecidas para indicar a duração de 200 e 600 milisegundos, respectivamente.

A partir do jQuery 1.4.3, uma palavra opcional nomeia uma função de easing(suavização) que pode ser usada. Funções easing especificam a velocidade que a animação transcorrerá nos disferentes pontos da animação. Há apenas implementações padrões na biblioteca jQuery, uma chamada swing, e outra que progride em um ritmo constante, chamada linear. Mais funções de easing estão disponíveis com o uso de plug-ins, o mais conhecido é jQuery UI suite.

Se fornecido, o callback é disparado uma vez que a animação tenha finalizado. Isto pode ser útil para amarrar animações diferentes juntas em sequência. O callback não envia qualquer argumento, mas this é definido apenas para elementos DOM serem animados. Se vários elementos são animados, é importante observar que o callback é executado para cada elemento correspondente, não apenas para a animação como um todo.

Podemos animar qualquer elemento, como uma simples imagem:

<div id="clickme">
  Click here
</div>
<img id="book" src="book.png" alt="" width="100" height="123" />
With the element initially hidden, we can show it slowly:
$('#clickme').click(function() {
  $('#book').show('slow', function() {
    // Animation complete.
  });
});

Observação adicional:

  • Todos efeitos jQuery, incluindo .show(), podem ser desligados globalmente pela configuração jQuery.fx.off = true, que efetivamente define a duration para 0. Para mais informações, consulte jQuery.fx.off.

Exemplos:

Exemplo: Anima todos os parágrafos escondidos, mostrando-os lentamente, finalizando a animação dentro de 600 milisegundos. 

<!DOCTYPE html>
<html>
<head>
  <style>
      p { background:yellow; }
      </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Show it</button>

      <p style="display: none">Hello  2</p>
<script>
    $("button").click(function () {
    $("p").show("slow");
    });
    </script>

</body>
</html>

Exemplo: Anima todas as divs escondidas, monstrando-as rapidamente em ordem, Finalizando cada animação dentro de 200 milisegundos. Quando uma animação é terminada, a proxima é inciada. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background:#def3ca; margin:3px; width:80px; 
  display:none; float:left; text-align:center; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <button id="showr">Show</button>
  <button id="hidr">Hide</button>
  <div>Hello 3,</div>

  <div>how</div>
  <div>are</div>
  <div>you?</div>
<script>
$("#showr").click(function () {
  $("div:eq(0)").show("fast", function () {
    /* use callee so don't have to name the function */
    $(this).next("div").show("fast", arguments.callee);
  });
});
$("#hidr").click(function () {
  $("div").hide(2000);
});

</script>

</body>
</html>

Exemplo: Mostra todos os elementos span e input com uma animação. Uma vez que a animação termina, altera o texto. 

<!DOCTYPE html>
<html>
<head>
  <style>
  span { display:none; }
  div { display:none; }
  p { font-weight:bold; background-color:#fcd; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Do it!</button>
  <span>Are you sure? (type 'yes' if you are) </span>
  <div>
    <form>
      <input type="text"  value="as;ldkfjalsdf"/>
    </form>
  </div>
  <p style="display:none;">I'm hidden...</p>
  
<script>
function doIt() {
  $("span,div").show("slow");
}
/* can pass in function name */
$("button").click(doIt);

$("form").submit(function () {
  if ($("input").val() == "yes") {
    $("p").show(4000, function () {
      $(this).text("Ok, DONE! (now showing)");
    });
  }
  $("span,div").hide("fast");
  /* to stop the submit */
  return false; 
});
</script>

</body>
</html>
