.size()

Categories: Miscellaneous > DOM Element Methods

.size() Retorna: Number

Descrição: Retorna o número de elementos no objeto jQuery.

  • adicionado na versão: 1.0.size()

O metódo .size() é funcionalmente equivalente a propiedade .length; porém, é preferível usar a propriedade .length porque ela não tem o overhead de uma chamada de função.

Dado uma lista simples e não ordenada na página:

<ul>
  <li>foo</li>
  <li>bar</li>
</ul>

Tanto .size() quanto .length identificam o número de itens:

alert( "Tamanho: " + $("li").size() );
alert( "Tamanho: " + $("li").length );

This results in two alerts:

Tamanho: 2

Tamanho: 2

Exemplo:

Conte as divs. Clique para adicionar mais.

<!DOCTYPE html>
<html>
<head>
  <style>
  body { cursor:pointer; min-height: 100px; }
  div { width:50px; height:30px; margin:5px; 
        float:left; background:blue; }
  span { color:red; }
 </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<span></span>
 <div></div>

<script>
$(document.body)
.click(function() { 
  $(this).append( $("<div>") );
  var n = $("div").size();
  $("span").text("Existem " + n + " divs. Clique para adicionar mais.");
})
// dispara-se o trigger de clique para iniciar
.click(); 
</script>

</body>
</html>
