.slideDown()

Categorias: Efeitos > Sliding

.slideDown( [ duration ], [ callback ] ) Retorna: jQuery

Descrição: Exibe os elementos selecionados com um movimento deslizante.

  • adicionado na versão: 1.0.slideDown( [ duration ], [ callback ] )

    durationUma string ou número que determina a duração da animação.

    callbackFunção que será chamada quando a animação completar.

  • adicionado na versão: 1.4.3.slideDown( [ duration ], [ easing ], [ callback ] )

    durationUma string ou número que determina a duração da animação.

    easingUma string indicando qual função easing(função pronta de animação) usar para a transição

    callbackFunção que será chamada quando a animação completar

O método .slideDown() anima a altura dos elementos selecionados. Isso faz com que os elementos abaixo destes sejam empurrados para baixo.

A duração tem que ser passada em milisegundos; valores altos indicam animações mais demoradas, não mais rápidas. As strigns 'fast' e 'slow' podem ser fornecidas para indicar a duração de 200 e 600 milisegundos, respectivamente. Se outra string for fornecida, ou se o parâmetro duration for omitido, é usado o valor padrão de 400 millisegundos.

A partir da jQuery 1.4.3, uma string opcional chamando uma função de easing pode ser usada. Funções de atenuação especificam a velocidade com que a animação progride em diferentes pontos dentro da animação. As únicas implementações na biblioteca jQuery como padrão, chamada swing, e a outra procede em ritmo constante se chama linear. Mais funções de easin estão disponíveis com o uso de plug-ins, como o notável suite jQuery UI.

Se fornecido, o callback é executado uma vez quando a animação é completada. Isso pode ser muito útil para fazer animações em sequência. O callback não envia nenhum argumento, mas this é setado para o elemento DOM que está sendo animado. Se vários elementos estão sendo animados, é importante notar que o callback é executado uma vez por elemento selecionado, não uma vez por animação como um todo.

Podemos animar qualquer elemento, como uma simples imagem:

<div id="clickme">
  Clique aqui
</div>
<img id="book" src="book.png" alt="" width="100" height="123" />

Com o elemento inicialmente escondido, podemos mostrá-lo lentamente:

$('#clickme').click(function() {
  $('#book').slideDown('slow', function() {
    // Animação completa.
  });
});

Notas adicionais:

  • Todos os efeitos jQuery, incluindo .slideDown(), pode ser desativado globalmente setando jQuery.fx.off = true, o que efetivamente define a duração para 0. Para mais informações veja jQuery.fx.off.

Exemplos:

Exemplo: Anima todos os divs para deslizarem para baixo e mostrarem-se em 600 milisegundos. 

<!DOCTYPE html>
<html>
<head>
  <style>
div { background:#de9a44; margin:3px; width:80px; 
height:40px; display:none; float:left; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Clique em mim!
<div></div>
<div></div>
<div></div>
<script>
$(document.body).click(function () {
if ($("div:first").is(":hidden")) {
$("div").slideDown("slow");
} else {
$("div").hide();
}
});

</script>

</body>
</html>

Demonstração:

Exemplo: Anima todos os inputs para deslizarem para baixo, Completando a animação em 1000 milisegundos. Uma vez que a animação termina, o visual do input do meio é alterado e ele recebe focus. 

<!DOCTYPE html>
<html>
<head>
  <style>
div { background:#cfd; margin:3px; width:50px; 
text-align:center; float:left; cursor:pointer;
border:2px outset black; font-weight:bolder; }
input { display:none; width:120px; float:left; 
margin:10px; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>Push!</div>
<input type="text" />
<input type="text" class="middle" />

<input type="text" />
<script>
$("div").click(function () {
$(this).css({ borderStyle:"inset", cursor:"wait" });
$("input").slideDown(1000,function(){
$(this).css("border", "2px red inset")
.filter(".middle")
 .css("background", "yellow")
 .focus();
$("div").css("visibility", "hidden");
});
});

</script>

</body>
</html>
