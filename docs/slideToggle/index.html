.slideToggle()

Categorias: Efeitos > Sliding

.slideToggle( [ duration ], [ callback ] ) Retorna: jQuery

Descrição: Exibe ou esconde os elementos selecionados com um movimento deslizante.

  • adicionado na versão: 1.0.slideToggle( [ duration ], [ callback ] )

    durationUma string ou número que determina a duração da animação.

    callbackFunção que será chamada quando a animação completar.

  • adicionado na versão: 1.4.3.slideToggle( [ duration ], [ easing ], [ callback ] )

    durationUma string ou número que determina a duração da animação.

    easingUma string indicando qual função easing(função pronta de animação) usar para a transição

    callbackFunção que será chamada quando a animação completar.

O metódo .slideToggle() anima a altura dos elementos selecionados. Isso faz com que os elementos abaixo deslizem para cima ou para baixo, mostrando ou revelando ou escondendo os itens. Se o elemento estiver inicialmente visivel, ele será escondido; se estiver escondido, ele será mostrado. A propriedade display é setada e restaurada quando precisar. Se um elemento tiver a propriedade de estilo display setado em inline, for escondido e mostrado, ele voltará a ter o valor inline em display. Quando a altura chegar a 0 depois de uma animação que ocultará o elemento, a propriedade de estilo display é setada para none para assegurar que o elemento não afetará mais o layout da página.

A partir da jQuery 1.4.3, uma string opcional chamando uma função de easing pode ser usada. Funções de atenuação especificam a velocidade com que a animação progride em diferentes pontos dentro da animação. As únicas implementações na biblioteca jQuery como padrão, chamada swing, e a outra procede em ritmo constante se chama linear. Mais funções de easing estão disponíveis com o uso de plug-ins, como o notável suite jQuery UI.

A duração tem que ser passada em milisegundos; valores altos indicam animações mais demoradas, não mais rápidas. As strigns 'fast' e 'slow' podem ser fornecidas para indicar a duração de 200 e 600 milisegundos, respectivamente. Se outra string for fornecida, ou se o parâmetro duration for omitido, é usado o valor padrão de 400 millisegundos.

Se fornecido, o callback é executado uma vez quando a animação é completada. Isso pode ser muito útil para fazer animações em sequência. O callback não envia nenhum argumento, mas this é setado para o elemento DOM que está sendo animado. Se vários elementos estão sendo animados, é importante notar que o callback é executado uma vez por elemento selecionado, não uma vez por animação como um todo.

Podemos animar qualquer elemento, como uma simples imagem:

<div id="clickme">
  Clique aqui
</div>
<img id="book" src="book.png" alt="" width="100" height="123" />

Vamos fazer .slideToggle() para ser chamada quando outro elemento for clicado:

$('#clickme').click(function() {
  $('#book').slideToggle('slow', function() {
    // Animation complete.
  });
});

Notas adicionais:

  • Todos os efeitos jQuery, incluindo .slideToggle(), pode ser desativado globalmente setando jQuery.fx.off = true, o que efetivamente define a duração para 0. Para mais informações veja jQuery.fx.off.

Exemplos:

Exemplo: Anima todos os parágrafos para deslizarem para cima ou para baixo, completando a animação em 600 milesegundos. 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { width:400px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Toggle</button>

  <p>
    This is the paragraph to end all paragraphs.  You
    should feel <em>lucky</em> to have seen such a paragraph in
    your life.  Congratulations!
  </p>
<script>
    $("button").click(function () {
      $("p").slideToggle("slow");
    });
</script>

</body>
</html>

Exemplo: Anima as divs entre os divisores com uma alternância que faz com que alguns apareçam e outros desapareçam. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background:#b977d1; margin:3px; width:60px; 
        height:60px; float:left; }
  div.still { background:#345; width:5px; }
  div.hider { display:none; }
  span { color:red; }
  p { clear: left; }</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div></div>
<div class="still"></div>
<div style="display:none;">
</div><div class="still"></div>
<div></div>
<div class="still"></div>
<div class="hider"></div>
<div class="still"></div>
<div class="hider"></div>
<div class="still"></div>
<div></div>
<p><button id="aa">Toggle</button> There have been <span>0</span> toggled divs.</p>
<script>
  $("#aa").click(function () {
    $("div:not(.still)").slideToggle("slow", function () {
      var n = parseInt($("span").text(), 10);
      $("span").text(n + 1);
    });
  });

</script>

</body>
</html>
