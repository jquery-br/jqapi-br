.stop()

Categories: Effects > Custom

.stop( [ clearQueue ], [ jumpToEnd ] ) Retorna: jQuery

Descição: Paraliza a animação que estiver sendo executada nos elesmentos alvo.

  • adicionado na versão: 1.2.stop( [ clearQueue ], [ jumpToEnd ] )

    clearQueueUm boolean indicando se a animação também deve ser removida da fila de execução. O padrão é false.

    jumpToEndUm boolean indicando se a animação atual deve ser executada completamente. O padrão é false.

Quando .stop() é chamado em um elemento, a animação atual (se existir alguma) é imediatamente paralizada. Se, por exemplo, um elemento está sendo ocultado com .slideUp() quando .stop() é chamado, o elemento agora continua a ser apresentado, porém será somente uma fração da sua altura original. Funções callback não são executadas .

Se mais de um método de animação é atribuída a um elemento, as animações posteriores são colocadas em fila de efeitos desse elemento. Estas animações não iniciam até que a primeira tenha sido completada. Quando .stop() é chamado, a animação seguinte na fila se inicia imediatamente. Se o parâmetro clearQueue é informado com o valor true, então o restante das animações da fila também serão removidas e numca serão executadas.

Se a propriedade jumpToEnd é informada com o valor true, a animação atual é paralizada, porém o elemento recebe imediatamente os valorespara cada propriedade CSS. Em nosso exemplo .slideUp() acima, o elemento será ocultado imediatamente. A função de callback é imediatamente executada se a mesma for determinada.

A utilidade do método .stop() é evidente quando precisamos animar um elemento em mouseenter e mouseleave:

<div id="hoverme">
  Passe o mouse
  <img id="hoverme" src="book.png" alt="" width="100" height="123" />
</div>

Podemos criar um belo efeito de fade sem o problema comum de várias animações em fila, acrescentando .stop(true, true) na sequência:

$('#hoverme-stop-2').hover(function() {
  $(this).find('img').stop(true, true).fadeOut();
}, function() {
  $(this).find('img').stop(true, true).fadeIn();
});

Animações podem ser paralizadas globalmente somente setando a propriedade $.fx.off para true. Quando isso é realizado todos os métodos de animação serão definidos para seu estado final quando forem chamados ao invés de exibir um efeito.

Exemplo:

Clique no botão Go para iniciar a animação, então clique no botão STOP para paralizar a animação na posição em que ela estiver. Outra opção é clicar em vários outros botões e ver que o stop somente paraliza o que estiver sendo executado no momento.

<!DOCTYPE html>
<html>
<head>
  <style>div { 
position: absolute; 
background-color: #abc;
left: 0px;
top:30px;
width: 60px; 
height: 60px;
margin: 5px; 
}
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="go">Go</button> 
<button id="stop">STOP!</button>
<button id="back">Back</button>
<div class="block"></div>
<script>
// Inicia a animação
$("#go").click(function(){
$(".block").animate({left: '+=100px'}, 2000);
});

// Paraliza a animação quando o botão é clicado
$("#stop").click(function(){
$(".block").stop();
});

// Inicia a animação em direção oposta.
$("#back").click(function(){
$(".block").animate({left: '-=100px'}, 2000);
});

</script>

</body>
</html>
