Categorias: Seletores > Formulários

seletor submit

adicionado na versão: 1.0jQuery(':submit')

Descrição: Seleciona todos os elementos do tipo submit.

O seletor :submit normalmente se aplica a botoões ou elementos do tipo input. Note que alguns navegadores tratam o elemento <button> como type="default" implicitamente enquanto outros (como o Internet Explorer) não.

Example:

Seleciona todos os elementos submit que são descendentes de um elemento td.
