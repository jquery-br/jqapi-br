.toggle()

Categoria: Efeitos > Básicos

.toggle( [ duration ], [ callback ] ) Retorna: jQuery

Descrição: Mostra ou oculta os elementos alvo.

  • adicionado na versão: 1.0.toggle( [ duration ], [ callback ] )

    durationUma string ou número determinando durante quanto tempo a animação será executada.

    callbackFunção a ser chamada ao completar a animação.

  • adicionada na versão: 1.4.3.toggle( [ duration ], [ easing ], [ callback ] )

    durationUma string ou número determinando durante quanto tempo a animação será executada

    easingUma string indicando qual função easing usar para a transição.

    callbackFunção a ser chamada ao completar a animação.

  • adicionada na versão: 1.3.toggle( showOrHide )

    showOrHideUm Boolean indicando quando mostrar ou ocultar os elementos.

Sem nenhum parâmetro, a função .toggle() simplesmente alterna a visibilidade dos elementos:

$('.target').toggle();

Os elementos alvos serão mostrados ou ocultos imediatamente, sem nenhuma animação, mudando a propriedade CSS display. Se o elemento é inicialmente mostrado, ele será oculto; se oculto será apresentado. A propriedade display é salva e restaurada quando necessário. Se um elemento tem display com valor inline, então ele será oculto e mostrado novamente de forma inline.

Quando é informada uma duração, .toggle() se torna um método de animação. O método .toggle() animates the width, height, and opacity of the matched elements simultaneously. When these properties reach 0 after a hiding animation, the display style property is set to none to ensure that the element no longer affects the layout of the page.

A duração é informada em milisegundos; Valores altos indicam animações mais lentas e não mais rápidas. Textos como 'fast' e 'slow' podem ser informados para indicar durações de 200 e 600 milisegundos, respectivamente.

Nota: A suíte de manipulação de eventos também possui um método chamado .toggle(). Qual será disparado dependerá dos argumentos informados.

A partir do jQuery 1.4.3, uma opção chamada de função easing passou a poder ser usada. Funções easing determinam a velocidade que a animação irá progredir em diferentes momentos dentro da mesma. A única implementação da easing no jQuery são somente as padrões, chamada swing, e outra que vem progredindo em constante ritmo, chamada linear. Mais funções easing estão disponíveis com o uso de plugins, mais notavelmente a jQuery UI suite.

Se for informado o callback é chamado assim que a animação é finalizada. Isto pode ser útil para unir animações diferentes em uma sequência. O callback não envia nenhum argumento, porém this é definido como o elemento DOM sendo animado. Se vários elementos estão sendo animados, é importante notar que o callback é executado uma vez por elemento correspondente e não para a animação como um todo.

Podemos animar qualquer elemento, por exemplo uma simples imagem:

<div id="clickme">
  Clique aqui
</div>
<img id="book" src="book.png" alt="" width="100" height="123" />

Vamos chamar .toggle() quando outro elemento for clicado:

$('#clickme').click(function() {
  $('#book').toggle('slow', function() {
    // Animação completa.
  });
});

Uma segunda versão do método aceita um parâmetro Boolean. Se o parâmetro é true, então os elementos alvos são apresentados; se false, os elementos são ocultados. Em essência seria a declaração: 

$('#foo').toggle(showOrHide);

que é equivalente a:

if ( showOrHide == true ) {
  $('#foo').show();
} else if ( showOrHide == false ) {
  $('#foo').hide();
}

Notas Adicionais:

  • Todos os efeitos da jQuery, incluíndo .toggle(), odem ser globalmente desativados pela configuração jQuery.fx.off = true, que efetivamente determina a duração como 0. Para mais informações, veja jQuery.fx.off.

Exemplo: Alterna todos os parágrafos. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Alternar</button>
<p>Olá</p>
<p style="display: none">Adeus</p>
<script>

$("button").click(function () {
$("p").toggle();
});
</script>

</body>
</html>
