. ToggleClass ()

Categorias: Atributos | Manipulação > Atributo Classe | CSS

. ToggleClass (className) Retorna: jQuery

Descrição: Adiciona ou remove uma ou mais classes de cada elemento no conjunto dos elementos combinados para serem usados, dependendo de cada presença da classe ou do valor do switch(true ou false)
(como uma chave que liga e desliga a classe).

  • Atualização na Versão: 1,0 toggleClass (NomedaClasse).

    NomedaClasse Um ou mais nomes de classe (separados por espaços) para ser alternado para cada elemento no conjunto a ser usado.

  • Atualização na Versão: 1,3 toggleClass (switch NomedaClasse).

    NomedaClasse Um ou mais nomes de classe (separados por espaços) para ser alternado para cada elemento no conjunto a ser usado.

    switch Um valor booleano do switch para determinar se a classe deve ser adicionadas ou removidas(true para adicionar e false para remover).

  • Atualização na Versão: 1,4 . toggleClass (função(índice, classe), [switch])

    função (índice, classe) Uma função que retorna nomes de classe a ser alternado no atributo classe de cada elemento no conjunto combinado. Recebe a posição de índice do elemento no conjunto e o valor da classe como argumentos.

    switch Um valor booleano para determinar se a classe deve ser adicionadas ou removidas.

Este método usa um ou mais nomes de classes como seu parâmetro. Na primeira versão, se um elemento do conjunto combinado de elementos já tem a classe, então a classe é removida; se um elemento não tem a classe, então ela é adicionada. Por exemplo, podemos aplicar toggleClass (). a uma simples <div> :

<div class="tumble">Algum Texto.</div>

A primeira vez que se aplica . $ ('div.tumble') toggleClass ('bounce') ,a definição da classe 'bounce' é removida da classe div.tumble, obtendo o seguinte:

<div class="tumble 'bounce'">Algum texto.</div>

A segunda vez que se aplica. $ ('div.tumble') toggleClass ('bounce') , a <div> da classe tumble retorma seu valor bounce:

<div class="tumble">Algum Texto.</div>

Aplicar . toggleClass ('bounce spin') para o mesmo <div> alterna entre <div bounce class="tumble spin"> e class="tumble"> <div> .

A segunda versão . toggleClass () usa um segundo parâmetro para determinar se a classe deve ser adicionadas ou removidas. Se o valor deste parâmetro é verdadeiro(true), então a classe é adicionada, se o valor forfalso(false) , a classe é removida. Em essência, a declaração fica asssim:

$ ('# Foo') toggleClass (NomedaClasse, True_ou_False).;

é equivalente a:

if (True_ou_False) {
$ ('# foo') addClass (NomedaClasse);.
} else {
$ ('# foo') removeClass (NomedaClasse);.
}

A partir de jQuery 1.4, a toggleClass() usa um método que permite indicar o nome da classe para ser alternada por passar em uma função.

$('div.foo').toggleClass(function() {
  if ($(this).parent().is('.bar')) {
    return 'happy';
  } else {
    return 'sad';
  }
});

Este exemplo irá alternar a classe happy para <div class="foo"> se seu elemento parent() tem uma classe bar , caso contrário, ele irá alternar para a classe sad.

Exemplo: Altere para a classe 'highlight' quando o parágrafo for clicado. 


<!DOCTYPE html>
<html>
<head>
  <style>

  p { margin: 4px; font-size:16px; font-weight:bolder; 
      cursor:pointer; }
  .blue { color:blue; }
  .highlight { background:yellow; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p class="blue">Clique para mudar o</p>
  <p class="blue highlight">highlight</p>
  <p class="blue">desses</p>
  <p class="blue">parágrafos</p>
<script>
    $("p").click(function () {
      $(this).toggleClass("highlight");
    });
</script>

</body>
</html>

Exemplo: Adicione a classe "highlight" e altere confome a quantidade de cliques, retire-a cada primeiro clique e adicione com o segundo. 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { margin: 4px; font-size:16px; font-weight:bolder; 
      cursor:pointer; }
  .blue { color:blue; }
  .highlight { background:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p class="blue">Clique para mudar o (<span>clicks: 0</span>)</p>
  <p class="blue highlight">highlight (<span>clicks: 0</span>)</p>
  <p class="blue">desses (<span>clicks: 0</span>)</p>
  <p class="blue">parágrafos (<span>clicks: 0</span>)</p>
<script>
var count = 0;
$("p").each(function() {
  var $thisParagraph = $(this);
  var count = 0;
  $thisParagraph.click(function() {
    count++;
    $thisParagraph.find("span").text('clicks: ' + count);
    $thisParagraph.toggleClass("highlight", count % 3 == 0);
  });
});

</script>

</body>
</html>
