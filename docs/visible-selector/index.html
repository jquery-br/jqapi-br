Seletor :visible

Categorias: Selectors > Visibility Filter

Seletor visible

adicionado na versão: 1.0jQuery(':visible')

Descrição: Seleciona todos os elementos que são visiveis.

Elementos podem ser considerados hidden por vários motivos:

  • Eles tem a propriedade do CSS display com valor none.
  • São elementos de formulário com type="hidden".
  • Sua altura e largura são explicitamente 0.
  • O elemento pai é hidden, então o elemento não é mostrado na página.

Elementos com visibility: hidden ou opacity: 0 são considerados visible, já que eles continuam consumindo espaço no layout. Durante animações que escondem um elemento, o elemento é considerado visible até o fim da animação. Durante uma animação para mostrar um elemento, o elemento é considerado visible desde o inicio da animação.

Exemplo:

Faz com que todos as divs visiveis fiquem amareças no click.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width:50px; height:40px; margin:5px; border:3px outset green; float:left; }
  .starthidden { display:none; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button>Show hidden to see they don't change</button>
  <div></div>
  <div class="starthidden"></div>
  <div></div>

  <div></div>
  <div style="display:none;"></div>
<script>
    $("div:visible").click(function () {
      $(this).css("background", "yellow");
    });
    $("button").click(function () {
      $("div:hidden").show("fast");
    });

</script>

</body>
</html>
